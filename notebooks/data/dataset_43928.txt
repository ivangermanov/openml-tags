The Medicare Inpatient Hospitals by Provider and Service dataset provides information on inpatient discharges for Original Medicare Part A beneficiaries by Inpatient Prospective Payment System hospitals. It includes information on the use, payment, and hospital charges for more than hospitals that received Inpatient Prospective Payment System payments. The data are organized by hospital and Medicare Severity Diagnosis Related Group. The Diagnosis Related Groups included in this dataset represent a significant portion of total Medicare Inpatient Prospective Payment System discharges.

The dataset includes information on the hospital provider, including the provider's organization name, state, city, state abbreviation, state Federal Information Processing Standard, zip code, and rural-urban commuting area. It also includes information on the Diagnosis Related Group, including the code and description. Additionally, the dataset provides information on the total discharges and average Medicare payment amount.

Keywords: Medicare, Inpatient Hospitals, Diagnosis Related Group, Hospital Charges, Payment System
