The Meta-Album Sports Actions Dataset is a collection of images covering various sports activities. This dataset is a subset of the one hundred sports dataset, which contains seventy-three different sports categories. The images are resized to one hundred twenty-eight by one hundred twenty-eight pixels using Open-CV.
