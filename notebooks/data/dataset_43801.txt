This Motorcycle Dataset contains information about used motorcycles. This data can be utilized for a multitude of purposes, such as price prediction, exemplifying the use of linear regression in Machine Learning. The columns in the given dataset are as follows: the motorcycle's name, its selling price, the year it was manufactured, the type of seller, the number of owners it has had, the total kilometers it has been driven, and its original ex-showroom price.

For datasets related to used cars, please refer to the Vehicle Dataset from Cardekho.

The dataset features include the motorcycle's name, selling price, year, seller type, owner, kilometers driven, and ex-showroom price.

Keywords: Motorcycle Dataset, Price Prediction, Linear Regression, Machine Learning, Used Motorcycles.
