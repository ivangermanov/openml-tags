This Motorcycle Dataset contains information about used motorcycles. This data can be utilized for a multitude of purposes, including predicting prices to demonstrate the application of linear regression in Machine Learning.

The columns present in the given dataset include the following: motorcycle name, selling price, year of manufacture, type of seller, number of owners, kilometers driven, and ex-showroom price.
