The Banana data set is an artificial collection of instances that belong to several clusters with a distinctive banana shape. There are two key attributes, namely X Axis and Y Axis, which correspond to the At One and At Two variables, respectively. The class label, which can take on values of negative one and one, represents one of the two distinct banana shapes present in the data set.

Features: Variable One, Variable Two, Class 

Keywords: Artificial Data Set, Clusters, Banana Shape, Attributes, Class Label, Variables
