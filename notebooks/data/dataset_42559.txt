The Inpatient Utilization and Payment Public Use File provides information on inpatient discharges for Medicare fee-for-service beneficiaries. This file includes information on utilization, payment, and hospital-specific charges for the more than three thousand United States hospitals that receive Medicare Inpatient Prospective Payment System payments. The Public Use File is organized by hospital and Medicare Severity Diagnosis Related Group and covers fiscal years.

The file provides detailed information on hospital-specific charges, including total discharges, average covered charges, average Medicare payments, and average total payments. Additionally, it includes information on the hospital's location, including the provider's name, street address, city, state, and zip code, as well as the hospital referral region description.
