Click Prediction Small Dataset

Tags: Chemistry, Derived, Life Science

This dataset contains a balanced version of click prediction data.

Features include:

* Click indicator,
* Number of impressions,
* URL hash,
* Advertisement identification number,
* Advertiser identification number,
* Depth information,
* Position details,
* Query identification number,
* Keyword identification number,
* Title identification number, 
* Description identification number, 
* User identification number.

Keywords: Click Prediction, Data Features, Advertisement, URL Hash, Identification Numbers
