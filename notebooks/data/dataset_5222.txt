**Schizo Dataset**

**Tags:** Author, Source, Unknown, Date unknown, Please cite

The Schizo dataset is a collection of eye-tracking data related to schizophrenia research. The dataset was published in an article titled "Schizophrenic Eye-Tracking Data" in the journal Biometrics.

Information About the Dataset:

The dataset has a nominal class type, and the class index is located at the last feature. This dataset contains various features, including identification information, target values, and several gain ratio measurements.

Features:

* Identification information
* Target values
* Sex
* Class
