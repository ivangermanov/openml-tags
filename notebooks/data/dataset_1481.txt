QSAR-TID-17061 Dataset 

This dataset contains QSAR data showing activity values of several compounds on drug target TID: 17061. It has multiple rows and numerous features, including IDs and class feature: Molecule ChEMBL ID and Median PX C Fifty.

The features represent FCFP Six One Hundred Twenty Four bit Molecular Fingerprints, which were generated from SMILES strings. These fingerprints were obtained using the Pipeline Pilot program, Dassault Systèmes BIOVIA. Generating fingerprints does not typically require missing value imputation as all bits are generated.

Dataset Description:

The dataset contains QSAR data showing activity values of several compounds on drug target TID: 17061. It has multiple rows and numerous features, including IDs and class feature: Molecule ChEMBL ID and Median PX C Fifty.

Compound Details:

The compound details include the molecule ChEMBL ID, which serves as a unique identifier for each compound. Additionally, the median PX C fifty value represents the activity of each compound on the drug target.

Molecular Fingerprints:

The molecular fingerprints were generated from SMILES strings using the Pipeline Pilot program, Dassault Systèmes BIOVIA. These fingerprints represent the chemical structure of each compound and are used to predict their biological activity.

Target Information:

The target information includes the preferred name, synonyms, and classification of the drug target. The preferred name is Alpha One B Adrenergic Receptor, and it belongs to the family of G protein coupled receptors.
