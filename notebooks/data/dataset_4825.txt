This dataset belongs to the "regression on numerical features" benchmark. It is one of forty-one drug design datasets formed using Adriana Code software. The molecules and outputs are taken from original studies.

The datasets are formed exactly from the original studies. The last attribute in each file is the target.

Original studies include Carbolenes, MTP2, Chang, Cristalli, Depreux, Doherty, Garrat2, Garrat, Heyl, Krystek, Lewis, Penning, Rosowsky, Siddiqi, Stevenson, Strupcz, Svensson, Thompson, Tsutumi, Uejling, Yokoyama1, and Yokoyama2.

Other original studies include Benzo Thirty-Two, Phenethyl One, PAH, PDGF-R, Phen, Topo Two One, Y Prop Four One, QSAR BR One, QSAR BR Two, QSARTOX, QSBR RW One, QSBR Y Two, QSBR Alks, QSFR DH LA, QSFSR One, QSFSR Two, and QSPR CMPX.

These datasets were taken from various studies including journals and books.

Features of this dataset are not human-readable.

Keywords: drug design, regression, numerical features, Adriana Code software, Carbolenes, MTP2, QSAR.
