California Dataset 
Tags: Machine Learning, Statistics 

This dataset is part of the tabular data benchmark and has been transformed in the same manner. It falls under the classification on numerical features benchmark category. 

The original source of this dataset can be found online, and users are requested to provide credit to the original source when utilizing this dataset.

The dataset comprises several key features, including Median Income, House Age, Average Rooms, Average Bedrooms, Population, Average Occupancy, Latitude, Longitude, and Price.
