The Year dataset is part of the tabular data benchmark, transformed in the same way as the original dataset. This dataset belongs to the regression on numerical features benchmark. The original source of this dataset is the UCI Machine Learning Repository.

The dataset features a total of ninety numerical features, which are used to predict the year of a song based on its audio features.

Please give credit to the original source if you use this dataset.
