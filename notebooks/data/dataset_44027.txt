Year Prediction MSD Dataset 

This dataset belongs to the Regression on Numerical Features benchmark and is part of the Tabular Data Benchmark, which was transformed in a similar manner. The original source of this dataset is the UCI Machine Learning Repository.

The Year Prediction MSD Dataset consists of various features, including timbre averages and covariances for songs. 

Please provide credit to the original source if you utilize this dataset.
