A collection of data sets used in the book "Analyzing Categorical Data," published by Springer-Verlag, New York, in 2003. The submission consists of a compressed file containing two versions of each of numerous data sets, along with a README file. Each data set is provided in comma-delimited ASCII (csv) format and Microsoft Excel (xls) format.

These data sets may be used freely for scientific, educational, and/or noncommercial purposes, provided suitable acknowledgment is given by citing the aforementioned reference. Further details concerning the book, including information on statistical software (such as sample S PLUS/R and SAS code), are available at the specified web site.

The dataset is classified as nominal. Quotes, single quotes, and backslashes were removed, and blanks were replaced with underscores in the original text. The features mentioned include political system, age, time of survey, schooling, region, and count.
