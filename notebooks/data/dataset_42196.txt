The Serie A Matches Dataset for Season 2015-2016

Tags: Economics, Geography

This dataset contains the Serie A matches for the 2015-2016 season.

Features:

* Identification Number
* Country Identification Number
* League Identification Number
* Season Description
* Stage Description
* Date of the Match
* Match API Identification Number
* Home Team API Identification Number
* Away Team API Identification Number
* Number of Goals Scored by the Home Team
* Number of Goals Scored by the Away Team
* List of Home Team Players
* List of Away Team Players
* Total Goals Scored During the Match
* Possession Percentage During the Match
* Bet365 Home Win Odds
* Bet365 Draw Odds
* Bet365 Away Win Odds

Keywords: Serie A, Football, Italy, Sports Data, Match Statistics
