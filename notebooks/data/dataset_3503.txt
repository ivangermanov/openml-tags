This dataset contains QSAR data showing activity values of several compounds on drug target TID: 20171. The features in the dataset are FCFP6 1024bit Molecular Fingerprints, which are a type of molecular descriptor used to characterize the chemical structure of compounds. These fingerprints were generated from SMILES strings using the Pipeline Pilot program, Dassault Systèmes BIOVIA.
