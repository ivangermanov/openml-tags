Fri C Two Dataset 

Tags: Chemistry, Life Science, Mythbusting One, Study One, Study Fifteen, Study Twenty, Study Forty-One, Study Seven, Study Eighty-Eight 

This dataset is a binarized version of the original data set. It converts the numeric target feature to a two-class nominal target feature by computing the mean and classifying all instances with a lower target value as positive ('Positive') and all others as negative ('Negative'). The resulting dataset has the following features: Oxygen One, Oxygen Two, Oxygen Three, Oxygen Four, Oxygen Five, Oxygen Six, Oxygen Seven, Oxygen Eight, Oxygen Nine, Oxygen Ten, Binary Class.
