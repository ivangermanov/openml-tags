The Anal Cat Data collection is a compilation of datasets used in the book Analyzing Categorical Data. This comprehensive collection consists of a zip file containing two versions of each of eighty-four datasets, along with a readme file. Each dataset is presented in both comma-delimited ASCII format and Microsoft Excel format.

Please note that these datasets may be utilized freely for scientific, educational, and non-commercial purposes, provided suitable acknowledgement is given by citing the original reference.

Further information regarding the book, including details on statistical software, can be found at the website.
