Football - Russian First League Football Extended

Tags: Context

This dataset contains games' statistics for the Russian First League, also known as the First Division or Football National League. The dataset spans across certain time periods, during which different statistics are available. It is interesting to explore potential links between in-game statistics and factual results, both past and forthcoming.

The data was generated using BeautifulSoup.

Acknowledgements are due to the various data sources available on the internet, which keep all protocols alive.

The dataset features include Half Time, Away Team, Home Goals, Away Goals, Total, Day, Month, Year, Full Date, Goal Kicks, Free Kicks, Out, Offsides, Blocked Shots, Possession, Fouls, Moments, Shots, Shots on Target, Half Time Whistle, Half Time Away, Corners, Yellow Cards, and Red Cards.

Keywords: Russian Football, First League, Game Statistics, In-Game Data, Factual Results.
