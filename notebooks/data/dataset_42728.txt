Airlines Departure Delay Prediction Dataset

This dataset is tagged as Airlines Departure Delay Prediction, a regression problem. The original data can be found at the Bureau of Transportation Statistics website. This is a processed version of the original data, designed to predict departure delay in seconds. A comma-separated values file of the raw data, spanning multiple years, can be found online.

This dataset consists of the first subset of the original data, in Attribute-Relation File Format. The features included in this dataset are departure delay, month, day of the month, day of the week, scheduled departure time, scheduled arrival time, unique carrier, origin, destination, and distance.

Keywords: Airlines, Departure Delay, Prediction, Regression, Transportation, Statistics
