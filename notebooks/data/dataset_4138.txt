Name: DBpedia YAGO ARFF

Tags: DBpedia with top most frequent YAGO types HMC dataset for type prediction

This dataset combines the power of DBpedia with the top most frequent YAGO types, creating a comprehensive HMC dataset ideal for type prediction tasks. The features of this dataset include ingoing and outgoing properties, which provide valuable insights for predicting types.

Keywords: DBpedia, YAGO, type prediction, HMC dataset, ingoing properties, outgoing properties
