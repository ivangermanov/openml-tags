**FOREX EURUSD Hourly High**

Tags: Finance, Forex, Forex High, Forex Hour, Machine Learning, Statistics

This dataset contains historical price data of the Forex EUR/USD currency pair from Dukascopy. Each instance represents one hourly candlestick. The data ranges from January to December, excluding weekends since Forex markets are closed during these periods. The timestamp feature is set to the Europe/Amsterdam timezone.

The class attribute indicates the direction of the mean high price for the following hour compared to the current hour. It is true if the mean high price increases in the next hour and false if it decreases or remains the same.

Please note that this dataset is intended for scientific purposes only, and realistic trading strategies should be based on close price predictions.

Attributes:

* Timestamp: The time of the current data point (Europe/Amsterdam)
* Bid Open: The bid price at the start of this time interval
* Bid High: The highest bid price during this time interval
* Bid Low: The lowest bid price during this time interval
* Bid Close: The bid price at the end of this time interval
* Bid Volume: The number of times the bid price changed within this time interval
* Ask Open: The ask price at the start of this time interval
* Ask High: The highest ask price during this time interval
* Ask Low: The lowest ask price during this time interval
* Ask Close: The ask price at the end of this time interval
* Ask Volume: The number of times the ask price changed within this time interval
* Class: Whether the average price will increase during the next interval

Keywords: Forex, EURUSD, Historical Data, Candlestick, Machine Learning, Finance
