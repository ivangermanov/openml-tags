The Bates Classification Dataset is categorized under Chemistry and Life Science. This dataset provides a description on GitHub, which can be accessed through the following link: https://github.com/slds-lmu/paper-2023-ci-for-ge. 

This dataset consists of various features, including response variable y and ninety-nine predictor variables, namely x one through x ninety-nine.
