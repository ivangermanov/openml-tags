Isolet Dataset: Binarized Version

The Isolet dataset is a binarized version of the original Isolet dataset. This version only considers instances with class labels One and Two from the original dataset.

The dataset features a wide range of attributes, including various acoustic measurements. These attributes are essential in understanding the characteristics of the Isolet dataset.
