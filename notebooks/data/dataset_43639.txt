Amazon Stock Data (October 2019 - July 2020) LATEST Tags: Context

This dataset contains the values of the stock of Amazon, which dates between October 2019 and July 2020. The dataset provides a comprehensive overview of the stock's performance during this period, making it an invaluable resource for anyone looking to analyze and make predictions about the company's stock.

The dataset contains seven different columns, each providing unique insights into the stock's behavior. These columns include the date, the opening value of the stock, the closing value of the stock, the highest and lowest values reached during the trading day, the adjusted closing value, and the volume of shares traded.

Acknowledgements

This dataset was scraped from Yahoo Finance, a reputable source of financial data. Yahoo Finance also provides an API that can be used to access this data, making it easy to integrate into your own projects and applications.

Features

Date, Open, High, Low, Close, Volume
