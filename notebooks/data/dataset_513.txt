Arsenic Female Lung Dataset Tags: 

The Arsenic Female Lung dataset contains information about various features related to arsenic exposure. 

The class type of this dataset is numeric and the class index is the last feature. 

The dataset includes features such as group classification, concentration levels, age of individuals, number of individuals at risk, and total events recorded.

Keywords: Arsenic, Female, Lung, Exposure, Numeric, Classification, Concentration
