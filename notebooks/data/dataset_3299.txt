Tennis Major Tournament Match Statistics - French Open Men 2013 

This dataset is part of a collection of eight files containing the match statistics for both women and men at the four major tennis tournaments of the year 2013. Each file has forty-two columns and a minimum of seventy-six rows. This file contains the statistics for the French Open Men's Edition for 2013.

Attribute Information 

Player One refers to the name of the first player, while Player Two refers to the name of the second player involved in the match. The Result attribute indicates the outcome of the match, where one represents a win for Player One and zero represents a loss. 

First Serve Percentage for Player One refers to the percentage of successful first serves made by Player One during the match. First Serve Won by Player One represents the number of points won by Player One through their successful first serves. Similarly, Second Serve Percentage for Player One refers to the percentage of successful second serves made by Player One, while Second Serve Won by Player One represents the number of points won through these successful second serves. 

Aces Won by Player One refers to the number of aces served by Player One during the match. Double Faults Committed by Player One represents the number of double faults made by Player One. Winners Earned by Player One refers to the total number of winners hit by Player One, while Unforced Errors Committed by Player One represents the total number of unforced errors made by Player One. 

Break Points Created by Player One refers to the number of break points created by Player One during the match, while Break Points Won by Player One represents the number of break points successfully converted into wins. Net Points Attempted by Player One refers to the total number of net points attempted by Player One, while Net Points Won by Player One represents the number of net points successfully won. 

Total Points Won by Player One refers to the total number of points won by Player One throughout the match. The attributes for Set One, Set Two, Set Three, Set Four, and Set Five Results for Player One refer to the scores achieved by Player One in each set. The Final Number of Games Won by Player One attribute represents the total number of games won by Player One throughout the match. 

The attributes for Player Two follow the same pattern, with the corresponding values representing the performance of Player Two during the match. The Round attribute indicates the round of the tournament at which the game was played.
