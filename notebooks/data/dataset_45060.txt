The Online Shoppers dataset consists of feature vectors belonging to sessions. The dataset was formed in a way that each session would belong to a different user in a one-year period to avoid any tendency towards a specific campaign, special day, user profile, or period.

Source: https://archive.ics.uci.edu/ml/datasets/Online Shoppers Purchasing Intention Dataset

Features: Bounce Rates, Exit Rates, Page Values, Special Day, Browser, Operating Systems, Visitor Type, Administrative Duration, Month, Region, Product Related, Product Related Duration, Informational, Traffic Type, Informational Duration, Weekend, Administrative, and class.

Keywords: online shopping, purchasing intention, user behavior, session data, feature vectors.
