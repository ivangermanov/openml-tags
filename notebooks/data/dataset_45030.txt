Pol Dataset: A Commercial Telecommunication Problem

This dataset belongs to the regression on numerical features benchmark. It describes a commercial application related to telecommunication.

The dataset contains numerous cases and forty-nine continuous attributes. It was sourced from a collection of regression datasets compiled by a researcher.

The original data in its initial format can be obtained from an online repository. 

Characteristics of this dataset include its large size and numerous continuous attributes.

Keywords: Telecommunication, Regression, Numerical Features, Commercial Application, Continuous Attributes
