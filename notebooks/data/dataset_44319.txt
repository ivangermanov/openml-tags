**Meta-Album Sports Actions Dataset (Extended)**

The 100-Sports dataset is a collection of sports images covering seventy-three different sports. Images are in .jpg format and were gathered from internet searches. The images were scanned with a duplicate image detector program and all duplicate images were removed. For Meta-Album, the dataset is preprocessed and images are resized into 128x128 pixels using Open-CV.

**Dataset Details**

The Meta-Album ID is Human Actions Sports. The Meta-Album URL is https://meta-album.github.io/datasets/SPT.html. The Domain ID is Human Actions and the Domain Name is Human Actions. The Dataset ID is Sports Actions and the Dataset Name is Sports Actions. The Short Description is 100 Sports Dataset.

**Dataset Information**

The dataset has seventy-three classes and 10,416 images. The keywords are human actions and sports. The data format is images and the image size is 128x128. The license for the original data release is CC0 1.0 Public Domain and the license URL is https://creativecommons.org/publicdomain/zero/1.0/. The license for the Meta-Album data release is also CC0 1.0 Public Domain and the license URL is https://creativecommons.org/publicdomain/zero/1.0/.

**Source Information**

The source is 100 Sports Image Classification and the source URL is https://www.kaggle.com/gpiosenka/sports-classification.
