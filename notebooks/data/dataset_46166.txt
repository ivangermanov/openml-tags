Name: Veteran Dataset
Tags: Dataset for Veterans

This dataset is from the R package Survival and it contains various features related to veterans. The treatment type is denoted by the feature Treatment, and the type of cell is represented by the feature Cell Type. The dataset also includes the feature Time, which indicates the duration of the study, and the feature Status, which shows the outcome of the study. Additionally, the dataset contains the feature Karnofsky Performance Status, which is a measure of the patient's overall health. The feature Diagnosis Time represents the time of diagnosis, and the feature Age represents the age of the veteran. Finally, the feature Prior indicates any prior illnesses or conditions.

Keywords: Veteran Dataset, Treatment, Cell Type, Time, Status, Karnofsky Performance Status, Diagnosis Time, Age, Prior Illnesses
