Reliance Stock Data July 2019 - July 2020
Tags: Context

This dataset contains the values of the stock of Reliance, which dates between July 15, 2019, and July 15, 2020.

The dataset contains seven different columns that include the date, the opening value of the stock, the closing value of the stock, volume, and a few other necessary things to make predictions.

The dataset was scraped from Yahoo Finance, and you can also use their API to access the data.

Features: Date, Open, High, Low, Close, Adjusted Close, Volume

Keywords: Reliance, Stock Data, Yahoo Finance, API, Predictions
