**Wimbledon Men 2013**

**Tags:** Tennis, Wimbledon, Men, 2013

This dataset is part of a collection of eight files containing match statistics for both women and men at the four major tennis tournaments of the year 2013. Each file has forty-two columns and a minimum of seventy-six rows. This file contains the statistics for the Wimbledon Men's Edition for 2013.

### Attribute Information

* Player One: Name of Player One
* Player Two: Name of Player Two
* Result: Result of the match (one if Player One wins, zero otherwise)
* First Serve Percentage for Player One: Percentage of first serves won by Player One
* First Serve Won by Player One: Number of first serves won by Player One
* Second Serve Percentage for Player One: Percentage of second serves won by Player One
* Second Serve Won by Player One: Number of second serves won by Player One
* Aces Won by Player One: Number of aces won by Player One
* Double Faults Committed by Player One: Number of double faults committed by Player One
* Winners Earned by Player One: Number of winners earned by Player One
* Unforced Errors Committed by Player One: Number of unforced errors committed by Player One
* Break Points Created by Player One: Number of break points created by Player One
* Break Points Won by Player One: Number of break points won by Player One
* Net Points Attempted by Player One: Number of net points attempted by Player One
* Net Points Won by Player One: Number of net points won by Player One
* Total Points Won by Player One: Number of total points won by Player One
* Set One Result for Player One: Result of set one for Player One
* Set Two Result for Player One: Result of set two for Player One
* Set Three Result for Player One: Result of set three for Player One
* Set Four Result for Player One: Result of set four for Player One
* Set Five Result for Player One: Result of set five for Player One
* Final Number of Games Won by Player One: Final number of games won by Player One
* First Serve Percentage for Player Two: Percentage of first serves won by Player Two
* First Serve Won by Player Two: Number of first serves won by Player Two
* Second Serve Percentage for Player Two: Percentage of second serves won by Player Two
* Second Serve Won by Player Two: Number of second serves won by Player Two
* Aces Won by Player Two: Number of aces won by Player Two
* Double Faults Committed by Player Two: Number of double faults committed by Player Two
* Winners Earned by Player Two: Number of winners earned by Player Two
* Unforced Errors Committed by Player Two: Number of unforced errors committed by Player Two
* Break Points Created by Player Two: Number of break points created by Player Two
* Break Points Won by Player Two: Number of break points won by Player Two
* Net Points Attempted by Player Two: Number of net points attempted by Player Two
* Net Points Won by Player Two: Number of net points won by Player Two
* Total Points Won by Player Two: Number of total points won by Player Two
* Set One Result for Player Two: Result of set one for Player Two
* Set Two Result for Player Two: Result of set two for Player Two
* Set Three Result for Player Two: Result of set three for Player Two
* Set Four Result for Player Two: Result of set four for Player Two
* Set Five Result for Player Two: Result of set five for Player Two
* Final Number of Games Won by Player Two: Final number of games won by Player Two
* Round: Round of the tournament at which the game is played

### Dataset Information

This dataset has missing values.
