Pigs Bayesian Network Sample Nine 

bnlearn Bayesian Network Repository reference 

This dataset contains four hundred forty-one nodes and five hundred ninety-two arcs with five thousand six hundred eighteen parameters. The average Markov blanket size is approximately three point sixty-six and the average degree is approximately two point sixty-eight. The maximum in-degree is two.

The features of this network include various nodes such as those related to astronomy and images.

This dataset is part of the bnlearn repository which provides a collection of Bayesian networks for research and experimentation.

Keywords: Bayesian Network, bnlearn, Astronomy, Images, Pigs
