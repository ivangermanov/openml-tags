Name: Bates Regression One Hundred

Tags: See GitHub repository for a description of this dataset.

Features: This dataset includes a dependent variable y and ninety-nine independent variables, including various features such as x one, x two, x three, and so on, up to x ninety-nine.

This dataset is designed for regression analysis and is part of a larger study on confidence intervals for generalized eigenvalues.

Keywords: regression analysis, confidence intervals, generalized eigenvalues, Bates regression.
