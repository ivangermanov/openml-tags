BNG Dataset 

Tags: Artificial, BNG, Chemistry, Life Science 

Features: 
This dataset contains an X box and Y box which are essential components for data analysis. The Width and High parameters provide valuable information about the dimensions of the data. The OnPix feature is also included. Furthermore, this dataset includes statistical measures such as X Bar, Y Bar, X Two Bar, Y Two Bar, XY Bar, X Two Y Bar, and XY Two Bar. Additionally, there are Edge features including X Edge, X Edge Gradient Y, Y Edge, and Y Edge Gradient X. Finally, this dataset also includes a classification feature.
