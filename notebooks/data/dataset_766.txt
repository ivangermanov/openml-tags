Fri C1 Five Hundred Fifty Regression Problem

This is a binarized version of the original data set. The numeric target feature has been converted into a two-class nominal target feature. This was done by calculating the mean and then classifying all instances with a lower target value as Positive and all others as Negative.

The dataset features include various ozone related variables.

This dataset is related to the fields of Chemistry and Life Science. It has also been used in various studies, including Mythbusting One, Study One, Study Fifteen, Study Twenty, Study Forty-One, and Study Eighty-Eight.
