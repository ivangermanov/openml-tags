**Wimbledon Women 2013**

This dataset is part of a collection of eight files containing match statistics for both women and men at the four major tennis tournaments of the year 2013. Each file has forty-two columns and a minimum of seventy-six rows. This file contains the statistics for the Wimbledon Women's Edition for 2013.

### Attribute Information

* Player One: The name of the first player.
* Player Two: The name of the second player.
* Result: The outcome of the match, where one indicates that Player One wins and zero indicates that Player Two wins.
* First Serve Percentage for Player One: The percentage of successful first serves made by Player One during the match.
* First Serve Won by Player One: The number of successful first serves won by Player One during the match.
* Second Serve Percentage for Player One: The percentage of successful second serves made by Player One during the match.
* Second Serve Won by Player One: The number of successful second serves won by Player One during the match.
* Aces Won by Player One: The number of aces served by Player One during the match.
* Double Faults Committed by Player One: The number of double faults committed by Player One during the match.
* Winners Earned by Player One: The number of winners earned by Player One during the match.
* Unforced Errors Committed by Player One: The number of unforced errors committed by Player One during the match.
* Break Points Created by Player One: The number of break points created by Player One during the match.
* Break Points Won by Player One: The number of break points won by Player One during the match.
* Net Points Attempted by Player One: The number of net points attempted by Player One during the match.
* Net Points Won by Player One: The number of net points won by Player One during the match.
* Total Points Won by Player One: The total number of points won by Player One during the match.
* Set One Result for Player One: The result of the first set for Player One.
* Set Two Result for Player One: The result of the second set for Player One.
* Set Three Result for Player One: The result of the third set for Player One.
* Set Four Result for Player One: The result of the fourth set for Player One.
* Set Five Result for Player One: The result of the fifth set for Player One.
* Final Number of Games Won by Player One: The final number of games won by Player One.

Similarly, there are attributes for Player Two, including their first serve percentage, second serve percentage, aces won, double faults committed, winners earned, unforced errors committed, break points created, break points won, net points attempted, net points won, total points won, and results for each set.

### Round
The round of the tournament at which the game was played.
