The chscase Vine Two dataset is a collection of data sets used in the book "A Casebook for a First Course in Statistics and Data Analysis". 

This dataset is part of a larger collection consisting of thirty-eight files, each representing a unique data set analyzed in the book. The file names correspond to the names used in the book. The data files are written in plain ASCII character text, with missing values represented by the letter "M". 

More information about the data sets and the book can be obtained through the Gopher protocol at the address swis.stern.nyu.edu, filed under Academic Departments and Research Centers, then Statistics and Operations Research, then Publications, then A Casebook for a First Course in Statistics and Data Analysis, and finally Welcome! Alternatively, it can also be accessed from the World Wide Web using a web browser, starting from the URL address http://www.stern.nyu.edu/SOR/Casebook.

These datasets may be used freely for scientific, educational, and/or non-commercial purposes, provided suitable acknowledgement is given.
