Attribute Descriptions:

The dataset contains features extracted from Change Cipher Spec TCP Acknowledgement, TLS Alert, and TCP Disconnect packets. These features include source and destination port numbers, sequence numbers, acknowledgment numbers, window sizes, and flags indicating various TCP states.

The Change Cipher Spec TCP Acknowledgement features include TCP source port, TCP destination port, TCP port, TCP stream, TCP segment length, TCP sequence number, TCP next sequence number, TCP acknowledgment number, TCP header length, TCP reserved flag, TCP nonce flag, TCP congestion window reduced flag, TCP ECN-echo flag, TCP urgent flag, TCP acknowledgment flag, TCP push flag, TCP reset flag, TCP syn flag, TCP fin flag, TCP window size value, TCP calculated window size, TCP window size scale factor, TCP checksum status, TCP urgent pointer, TCP options nop, TCP option kind, TCP option length, TCP options timestamp value, TCP time delta, and message order.

The TLS Alert features include TCP source port, TCP destination port, TCP port, TCP stream, TCP segment length, TCP sequence number, TCP next sequence number, TCP acknowledgment number, TCP header length, TCP reserved flag, TCP nonce flag, TCP congestion window reduced flag, TCP ECN-echo flag, TCP urgent flag, TCP acknowledgment flag, TCP push flag, TCP reset flag, TCP syn flag, TCP fin flag, TCP window size value, TCP calculated window size, TCP window size scale factor, TCP checksum status, TCP urgent pointer, TCP options nop, TCP option kind, TCP option length, TCP options timestamp value, TCP time delta, TLS record content type, TLS record length, TLS alert message level, TLS alert message description, and message order.

The TCP Disconnect features include TCP source port, TCP destination port, TCP port, TCP stream, TCP segment length, TCP sequence number, TCP next sequence number, TCP acknowledgment number, TCP header length, TCP reserved flag, TCP nonce flag, TCP congestion window reduced flag, TCP ECN-echo flag, TCP urgent flag, TCP acknowledgment flag, TCP push flag, TCP reset flag, TCP syn flag, TCP fin flag, TCP window size value, TCP calculated window size, TCP window size scale factor, TCP checksum status, TCP urgent pointer, TCP options nop, TCP option kind, TCP option length, TCP options timestamp value, TCP time delta, and message order.
