Stylized Meta Album Airplane Dataset with Various Airplane Models

This dataset features a collection of airplane models, each with its unique characteristics and stylized representations. The dataset includes a category classification, file name, and file path for each airplane model. Additionally, it provides information on the style and style name associated with each model, along with the style path and stylized path.

Features:
Category, File Name, File Path, Style, Style Name, Style Path, Stylized Path

Tags: Airplanes dataset with different airplane models (stylized)

Keywords: Airplane Models, Stylized Dataset, Category Classification, File Information, Style Representation
