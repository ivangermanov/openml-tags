IMDb Ratings Dataset

Tags: Earth Science, Life Science

This dataset contains IMDb ratings and votes information for movies having an original title, which is useful for creating a top-rated movies recommender system.

The dataset includes the following columns:

* Title ID: an alphanumeric unique identifier of the title
* Title: the original title of the movie
* Average Rating: a weighted average of all the individual user ratings
* Number of Votes: the total number of votes the title has received

Acknowledgement:
We would like to express our gratitude to IMDb for providing detailed information about numerous movies.

Inspiration:
Using this dataset, one can identify the highest rated movies currently and gain insights into what makes a movie successful. The dataset provides a valuable resource for film enthusiasts and researchers alike, enabling them to analyze and understand the factors that contribute to a movie's popularity.

Features:
Title ID, Title, Average Rating, Number of Votes

Keywords: IMDb ratings, movies, recommender system, film analysis, popularity metrics, user reviews.
