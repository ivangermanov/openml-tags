The Inpatient Utilization and Payment Public Use File provides information on inpatient discharges for Medicare fee-for-service beneficiaries. This file includes details on utilization, payment (including total payment and Medicare payment), and hospital-specific charges for the numerous United States hospitals that receive Medicare Inpatient Prospective Payment System payments.

The file is organized by hospital and Medicare Severity Diagnosis Related Group, and it covers a range of fiscal years. It offers a wealth of information about healthcare providers, including their name, street address, city, state, and zip code. Additionally, it provides data on hospital referral regions, as well as the total number of discharges.

Furthermore, the file includes financial data, such as the average covered charges, average total payments, and average Medicare payments.
