Trump Tweets 

Tags: Computer Systems, Machine Learning 

This dataset collects recent tweets posted by the forty-fifth President of the United States of America. We utilized the Tweepy Python package to access the Twitter Application Programming Interface for collecting this data. Our inspiration stems from studying the subjects of recent tweets posted by the forty-fifth President of the United States of America and performing various Natural Language Processing tasks on this data source.

Features include the unique identifier, username, location of the user, description of the user, creation date of the user account, number of followers, number of friends, number of favorite tweets, verification status, date of posting, tweet content, hashtags used, source of the tweet, number of retweets, and number of likes.

Keywords: tweets, president, machine learning, computer systems, natural language processing, social media, data collection.
