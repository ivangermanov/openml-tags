Trump Tweets Tags: Context

We collect recent tweets of the President of the United States of America. The data is collected using a Python package called Tweepy to access the Twitter Application Programming Interface. 

Inspiration for this dataset comes from studying the subjects of recent tweets of the President of the United States of America, and performing various Natural Language Processing tasks on this data source.

Features of this dataset include the tweet identification, username, user location, user description, user creation date, number of user followers, number of user friends, number of user favourites, user verification status, date of the tweet, tweet text, hashtags, source of the tweet, number of retweets, number of favourites, and whether the tweet is a retweet.

Keywords: Twitter, President, Natural Language Processing, Tweepy, Python
