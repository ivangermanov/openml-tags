Content: This dataset brings you the Twitter responses made with the hashtag #TheSocialDilemma after watching the eye-opening documentary "The Social Dilemma" released on an over-the-top platform, Netflix, on September 9th, 2020. The dataset was extracted using Twitter API, consisting of nearly ten thousand five hundred twenty-six tweets from Twitter users all over the globe.

The dataset includes information about the user, such as their username, location, and description, as well as details about their account, including the number of followers, friends, and favorites. Additionally, the dataset includes the date and time the tweet was created, the actual text of the tweet, any other hashtags posted in the tweet, the utility used to post the tweet, and whether the tweet has been retweeted.

The sentiment of the tweet is the target variable, which indicates whether the tweet has a positive, neutral, or negative sentiment.

This dataset can be used to dive into the subjects that use this hashtag, look at the geographical distribution, evaluate sentiments, and look for trends.

Features: username, location, description, account creation date, number of followers, number of friends, number of favorites, verification status, date and time, tweet text, hashtags, source, retweet status, sentiment
