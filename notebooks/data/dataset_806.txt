Fri C Three Thousand Fifty Dataset 

Tags: Binarized Regression Problem, Chemistry, Life Science, Mythbusting One, Study One, Study Fifteen, Study Twenty, Study Forty-One, Study Seven 

This dataset is a binarized version of the original data set. The numeric target feature has been converted into a two-class nominal target feature by calculating the mean and classifying all instances with a lower target value as Positive and all others as Negative. 

The dataset contains various features including ozone one, ozone two, ozone three, and so on up to ozone fifty, along with a binary class feature.
