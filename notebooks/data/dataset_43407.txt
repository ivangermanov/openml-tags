Problem Statement 

The following data analysis of marketing campaigns is a part of the assignment for Data Science Intern at Merkle Sokrati. The primary objectives of this task are to carry out exploratory data analysis and build machine learning models to evaluate the insights automatically. Additionally, prepare a comprehensive summary of the analysis and present it in a professional-looking deck.

Content Marketing Campaigns Data Set 

This data set contains information from October nineteen to July twenty, sourced from Google and Facebook campaigns. It showcases the performance of various age groups across different dimensions. All key fields, including platform, type, medium, sub channel, audience, and creative, have already been mapped to the data.

Platform 
Marketing platforms on which campaigns are running, primarily comprising Google Ads and Facebook Ads.

Type 
The type of campaign; in this data, only Google search and Facebook conversion campaigns have been considered.

Medium 
The method through which we connect with people in our marketing campaigns, either via specific keywords or creatives.

Sub Channel 
Subchannel refers to the specific keywords targeted under Google Search, and the subchannel targeted in Facebook.

Audience 
Multiple types of audiences are being targeted in different campaigns, encrypted as Audience One, Two, Three, and so forth.

Creative 
This refers to the type of image, video, or carousel used in Facebook ads.

Data Set Features 

Date, Product, Phase, Campaign Platform, Campaign Type, Communication Medium, Subchannel, Audience Type, Creative Type, Creative Name, Device, Age, Spends, Impressions, Clicks, Link Clicks
