Attribute Descriptions:

This dataset contains attributes related to TCP packets and TLS alerts.

CCS Zero: This section contains attributes related to the first Change Cipher Spec TCP acknowledgement. It includes attributes such as source port, destination port, port, stream, segment length, sequence number, next sequence number, acknowledgement number, header length, reserved flags, nonce flags, congestion window reduced flags, ECN echo flags, urgent flags, acknowledgement flags, push flags, reset flags, syn flags, fin flags, window size value, calculated window size, window size scale factor, checksum status, urgent pointer, NOP options, option kind, option length, timestamp value, time delta, and message order.

TLS Zero: This section contains attributes related to the first TLS alert. It includes attributes such as source port, destination port, port, stream, segment length, sequence number, next sequence number, acknowledgement number, header length, reserved flags, nonce flags, congestion window reduced flags, ECN echo flags, urgent flags, acknowledgement flags, push flags, reset flags, syn flags, fin flags, window size value, calculated window size, window size scale factor, checksum status, urgent pointer, NOP options, option kind, option length, timestamp value, time delta, record content type, record length, alert message level, alert message description, and message order.

DISC Zero and DISC One: These sections contain attributes related to the first and second TCP disconnects respectively. They include attributes similar to those found in the CCS Zero section.
