DBLP QuAD is a scholarly question answering dataset built over the DBLP knowledge graph. This dataset can also be accessed at the following online locations: a repository on Zenodo and a dataset on Hugging Face. Additionally, the research paper associated with this dataset is available on arXiv. The reference DBLP Knowledge Graph dump in NT format can be found on Zenodo as well.

The dataset features a unique identifier, the type of query, the original question, a paraphrased version of the question, the actual query, a template identifier, and information about entities and relations. It also includes temporal data and a held-out section.

Keywords: scholarly question answering, knowledge graph, dataset, DBLP, question answering, research paper.
