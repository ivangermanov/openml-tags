Football Striker Performance Dataset

Tags: Computer Systems, Machine Learning

The primary objective of this dataset is to provide a comprehensive platform to evaluate and compare the performance of various football players, specifically strikers, across different European leagues, within a concise set of thirty columns.

This dataset comprises an array of performance indicators, including goals scored, assists provided, minutes played, and games participated in, for football strikers over the past five years.

Acknowledgments

The data has been meticulously extracted from Transfermarkt website.

Features:

Age, Current Club, Current League, Foot, Height, Name, Nationality, Position

Keywords: Football, Performance, Strikers, European Leagues, Computer Systems, Machine Learning
