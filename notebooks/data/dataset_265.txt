Name: BNG (glass)

Tags: Features

This dataset focuses on the analysis of glass samples. It includes various features that characterize the composition and properties of different types of glass. The features examined in this dataset are:

Refractive Index (RI)
Sodium (Na)
Magnesium (Mg)
Aluminum (Al)
Silicon (Si)
Potassium (K)
Calcium (Ca)
Barium (Ba)
Iron (Fe)
Type

These features provide comprehensive information about the chemical and physical attributes of glass samples, allowing for detailed classification and analysis of different glass types.

Keywords: glass analysis, chemical composition, refractive index, elemental analysis, glass classification
