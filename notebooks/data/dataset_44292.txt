**Meta Album Insects2 Dataset (Mini)**

The Pest Insects dataset was originally created as a large-scale benchmark dataset for Insect Pest Recognition. It contains a vast collection of images belonging to multiple categories. The dataset also features a hierarchical taxonomy, where insect pests that mainly affect one specific agricultural product are grouped into the same upper-level category.
