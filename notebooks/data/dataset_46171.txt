Name: Tumor Dataset
Tags: Dataset Tumor from R Package PAMM Tools

This dataset contains information about tumors and is part of the PAMM Tools package in R. The features included in this dataset are the patient's status, Charlson score, age, whether they received a transfusion, any complications that arose, the presence of metastases, whether a resection was performed, the time frame, and the patient's sex.

Keywords: Tumor, PAMM Tools, Charlson Score, Transfusion, Metastases, Resection
