Weather in Istanbul Data

Tags: Context

This project is trained by obtaining weather data from Istanbul. The content of this dataset includes daily weather observations, rainfall, sunrise time, sunset time, moonrise time, moonset time, average wind speed, average humidity, and average pressure in Istanbul.

The dataset is sourced from daily observations taken from the World Weather Online website. The inspiration behind this project is to estimate tomorrow's weather using a binary classification method.

The features of this dataset include Date and Time, Weather Condition, Rainfall, Maximum Temperature, Minimum Temperature, Sunrise Time, Sunset Time, Moonrise Time, Moonset Time, Average Wind Speed, Average Humidity, and Average Pressure.

Keywords: Weather Forecast, Istanbul, Daily Observations, Binary Classification, Weather Data
