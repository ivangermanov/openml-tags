Page Blocks Dataset 

Tags: Chemistry, Life Science, Myth Busting, Study 

This dataset is a binarized version of the original dataset. The multi-class target feature has been converted into a two-class nominal target feature, where the majority class is labeled as Positive and all other classes are labeled as Negative. 

The dataset features include Height, Length, Area, Eccentricity, Percentage of Black Pixels, Percentage of Pixels with And Operation, Mean Transparency Ratio, Number of Black Pixels, Number of Black and White Pixels, White Balance Transparency Ratio, and Binary Class.
