California Dataset

This dataset is used in the Tabular Data Benchmark and has been transformed in the same way. It belongs to the Classification on Numerical Features benchmark.

The original source of this dataset is a California Housing dataset, which can be found online. Please give credit to the original source if you use this dataset.

The dataset features include Median Income, House Age, Average Rooms, Average Bedrooms, Population, Average Occupation, Latitude, Longitude, and Price Above Median.
