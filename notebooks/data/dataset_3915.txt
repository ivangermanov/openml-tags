The QSAR-TID-11109 dataset contains QSAR data showing activity values of several compounds on drug target Matrix Metalloproteinase Three. This dataset has one thousand nine hundred seventy-six rows and one thousand twenty-six features, including IDs and class feature.

Features include Molecular Fingerprints, which were generated from SMILES strings using the Pipeline Pilot program, Dassault Systemes BIOVIA.

This dataset was obtained from the MetaQSAR project in September two thousand fifteen.

Matrix Metalloproteinase Three is also known as Stromelysin One, Matrix Metalloproteinase, MMP Three, MMP, SL One, STMY One.

It belongs to the enzyme/protease/metallo protease/metallo protease MAM clan/metallo protease M Ten A subfamily classification.

Matrix Metalloproteinase Three is involved in various biological processes, including immune system process, proteolysis, extracellular matrix disassembly, extracellular matrix organization, collagen catabolic process, positive regulation of protein-containing complex assembly, cellular response to reactive oxygen species, innate immune response, cellular response to UV-A, cellular response to nitric oxide, regulation of neuroinflammatory response, response to amyloid-beta, negative regulation of reactive oxygen species metabolic process.

It is also associated with various diseases, including cancer, arthritis, and cardiovascular disease.
