Data Sets for Regression Models for Time Series Analysis. This dataset contains monthly numbers of tourist arrivals in Cyprus, covering a period from January nineteen seventy-nine to December two thousand. 

Information about the dataset: 
The class type of this dataset is numeric, and there is no specific class index. 
The dataset consists of three features: year, month, and tourist arrivals.
