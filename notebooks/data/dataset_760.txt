Anal Cat Data UK Train Accident Dataset 

Tags: Chemistry, Life Science, Mythbusting, Study 

This dataset is a binarized version of the original dataset. The numeric target feature has been converted into a two-class nominal target feature by calculating the mean and categorizing all instances with a lower target value as Positive and all others as Negative. 

The features included in this dataset are Year, Train Kilometers, Percentage Mark One, Accidents, Signal Passed At Danger Preventable, Other Preventable, Non Preventable, Grouped Year, Grouped Accidents, Grouped Signal Passed At Danger, Grouped Other, Grouped Non, Grouped Train Kilometers, Fatalities, Signal Passed At Danger Fatalities, and Other Fatalities, along with a Binary Class.
