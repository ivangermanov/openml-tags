Name: Non Alcoholic Fatty Liver Disease Dataset One
Tags: Dataset from R Package Survival Analysis

This dataset is known as the Non Alcoholic Fatty Liver Disease Dataset One and it is part of the R package for Survival Analysis. The dataset contains various features that are used to analyze the survival rates of patients. These features include the patient's age, weight, height, body mass index, status, sex, and time.
