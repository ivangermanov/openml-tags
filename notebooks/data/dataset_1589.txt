**Svmguide Three**

Tags: Chemistry, Life Science

This dataset is from the LIBSVM data repository. The original data was obtained from someone working in the car industry in Germany.

The dataset has been preprocessed from its original form. It contains twenty-one features, including attributes one through twenty-one, and a class label. 

Please cite this dataset as follows: Dataset from the LIBSVM data repository.
