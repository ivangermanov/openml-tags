Hyper Plane Concept Drift Study Dataset 

This dataset is characterized by the presence of concept drift, which refers to changes in the underlying distribution of the data over time, leading to a degradation in the performance of machine learning models.

The dataset is part of a larger study, specifically Study Sixteen, which aims to investigate the effects of concept drift on various machine learning algorithms.

The dataset consists of ten attributes, including Att One, Att Two, Att Three, Att Four, Att Five, Att Six, Att Seven, Att Eight, Att Nine, and Att Ten, which are used to predict the class label.

Keywords: Concept Drift, Machine Learning, Study Sixteen, Synthetic Data
