This dataset is a subset of the KDD Cup Two Thousand Twelve Track Two data, specifically created for the research paper Gate: Gated Additive Tree Ensemble for Tabular Classification and Regression. The data was retrieved from Dropbox.

**Important Note:** Please thoroughly read the Kaggle dataset description prior to utilizing this dataset.

This dataset primarily consists of identification codes that require lookup in separate files not available on OpenML, which were not utilized as part of the benchmark in the aforementioned research paper.

Features include impression, URL hash, advertisement identification, advertiser identification, depth, position, query identification, keyword identification, title identification, description identification, and user identification, alongside the target variable.

Keywords: chemistry, life science, tabular classification, regression, gated additive tree ensemble, dataset, kdd cup.
