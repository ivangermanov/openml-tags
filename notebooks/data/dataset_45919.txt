Padding Attack Dataset Facebook

Tags: Bleichenbacher Padding Attack

This dataset was created on December fifth, two thousand twenty-three, with a server named Facebook.

Attribute Names:

The dataset contains various attributes related to TCP packets, including source port, destination port, port, stream, length, sequence number, next sequence number, acknowledgment number, header length, flags, window size value, window size, window size scale factor, checksum status, urgent pointer, options, and timestamp.

The dataset also includes attributes related to the order of packets and the presence of certain bytes in the message.

Vulnerable Classes:

The dataset includes classes that are vulnerable to padding attacks, including packets with incorrect PKCS#1 padding, packets with incorrect message length, packets with incorrect first byte, and packets with incorrect second byte.

Features:

The dataset includes a wide range of features related to TCP packets, including source port, destination port, port, stream, length, sequence number, next sequence number, acknowledgment number, header length, flags, window size value, window size, window size scale factor, checksum status, urgent pointer, options, timestamp, and order.
