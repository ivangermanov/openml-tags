Zoo Dataset 

Tags: Chemistry, Life Science, Myth Busting, Study 

The binarized version of this dataset converts the multi-class target feature into a two-class nominal target feature by relabeling the majority class as Positive and all others as Negative. This conversion was originally done by removing multi-class information. 

Features: Animal Characteristics, Presence of Hair, Feathers, Egg-Laying Ability, Milk Production, Ability to Fly, Aquatic Nature, Predator Status, Tooth Structure, Vertebral Column, Respiratory System, Venom Presence, Fin Characteristics, Leg Structure, Tail Presence, Domestication, Size Comparable to Cats, Classification Binary Class 

Keywords: Zoo Dataset, Animal Characteristics, Classification, Biology, Chemistry
