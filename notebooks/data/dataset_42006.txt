Web Evaluations Tags: Web Services Evaluations

This table contains information about Web Evaluations, which is a dataset that stores evaluations of web services. The features of this dataset include Identification, User Identification, Web Service Identification, Response Time, Throughput, True Positive, True Positive Normalized, Response Time Normalized, User Trust Level, User Throughput Trust, and User Response Time Trust.
