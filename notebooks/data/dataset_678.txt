This dataset dump contains various datasets from the book Visualizing Environmental Data. The datasets were created using the data.dump function and can be easily restored into an S environment using the data.restore function.

Each dataset in this collection has a unique name, which corresponds to its respective entry in the book's index. For instance, one of the datasets is named Barley. To find more information about the Barley dataset, simply look up the entry "Data, Barley" in the index.

The file containing these datasets is located at ../data/visualizing/environmental.csv.

Information about the Environmental Dataset:
Class Type: Numeric
Class Index: None Specific

Features of the Environmental Dataset include Ozone Levels, Radiation Measurements, Temperature Readings, and Wind Speed.

Keywords: Environmental Data, Visualization, Datasets, Statistical Analysis, Climate Measurements
