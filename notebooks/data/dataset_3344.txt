Internet Firewall Data Set

Tags: Images, Life Science

This dataset was collected from internet traffic records on a university's firewall.

The Internet Firewall Data Set contains twelve features in total. The Action feature is used as a class, which has four categories: Allow, Action, Drop, and Reset Both.

Attribute Information:

* Source Port
* Destination Port
* NAT Source Port
* NAT Destination Port
* Action
* Bytes
* Bytes Sent
* Bytes Received
* Packets
* Elapsed Time (in seconds)
* Packets Sent
* Packets Received
