The United States Postal Service dataset has been binarized and modified to only include instances with class labels Six and Nine from the original dataset. These labels have been encoded as Zero for the original class Six and One for the original class Nine.

This dataset is tagged with Social Media and Statistics. The features included in this dataset are numerous, but some notable ones are integers and doubles representing various characteristics of postal service data. 

Keywords: USPS, Social Media, Statistics, Binary Classification, Dataset
