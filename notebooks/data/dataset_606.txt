Anal Cat Data Olympic Dataset 

Tags: Chemistry, Life Science, Myth Busting, Study, Trivial 

This dataset is a binarized version of the original data set. It converts the numeric target feature to a two-class nominal target feature by computing the mean and classifying all instances with a lower target value as Positive and all others as Negative. The dataset includes features such as Country, number of gold medals won in the year two thousand, number of silver medals won in the year two thousand, number of bronze medals won in the year two thousand, total number of medals won in the year two thousand, population, number of athletes, gross domestic product, total number of medals won in nineteen ninety-six, logarithmic values for nineteen ninety-six, logarithmic values for population, and logarithmic values for gross domestic product.

Keywords: Olympics, Medals, Country, Population, Athletes, Gross Domestic Product
