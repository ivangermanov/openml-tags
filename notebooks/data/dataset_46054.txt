Name: Stylized Meta Album Plant Network Style Mini

Tags: Plants Dataset with Different Species of Plants (Stylized)

This dataset features a collection of plant species, each with its unique characteristics and stylized representations. The dataset includes a category classification for each plant, along with its file name and file path. Additionally, the dataset provides information on the style and style name used to create the stylized representation, as well as the path to the stylized image.
