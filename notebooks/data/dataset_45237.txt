**Link Bayesian Network**

This Bayesian network repository reference is a discrete very large dataset.

The Link Bayesian Network consists of seven hundred twenty-four nodes and one thousand one hundred twenty-five arcs. It has fourteen thousand two hundred eleven parameters, with an average Markov blanket size of approximately five and an average degree of roughly three.

This dataset has been used extensively in machine learning and manufacturing applications.

**Features**

Some notable features include Node Twenty-Four Degree G, Node Sixty-Nine Alpha M, Node Forty-Nine Degree M, and many others.

**Keywords:** Bayesian Network, Machine Learning, Manufacturing, Discrete Data, Very Large Dataset
