Compass dataset is categorized under Computer Systems and Physical Sciences, and it is part of the classification on categorical and numerical features benchmark. The dataset has been transformed and is available on GitHub at tabular-benchmark repository.

This dataset was originally sourced from ProPublica's COMPAS analysis, specifically from the cox-violent-parsed.csv file. Therefore, credit should be given to the original source if this dataset is utilized.

The Compass dataset comprises various features, including Sex, Age, Age Category, Race, Juvenile Felony Count, Juvenile Misdemeanor Count, Juvenile Other Count, Prior Counts, Days Between Screening and Arrest, Days from COMPAS Assessment, Charge Degree, Decile Score, Score Text, Type of Assessment, Decile Score, Score Text, End, and Is Recidivism.

Keywords: Compass dataset, Computer Systems, Physical Sciences, Classification Benchmark, Categorical Features, Numerical Features.
