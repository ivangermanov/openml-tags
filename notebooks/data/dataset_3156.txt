The Flora dataset is a chemistry-related dataset tagged with machine learning and AutoML challenge two thousand fourteen. The original task for this dataset was regression. To obtain the test and validation sets, one can visit the ChaLearn website.

This dataset consists of various features, including numerous chemical properties and characteristics. There are over one hundred ninety-nine thousand features in total.

The ChaLearn website provides access to the necessary data for working with this dataset.

Keywords: Flora, chemistry, machine learning, AutoML, regression, ChaLearn.
