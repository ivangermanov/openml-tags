Traffic Violations Dataset

Context:
This dataset contains traffic violation information from all electronic traffic violations issued in the County. To protect privacy, any information that can be used to uniquely identify the vehicle, the vehicle owner, or the officer issuing the violation will not be published.

Acknowledgements:
The source of the original dataset is available at https://catalog.data.gov/dataset/traffic-violations.

Inspiration:
The original dataset was too time-consuming to perform several basic tasks, so it was shuffled and the first portion was taken to create a more manageable dataset.

Features:
Date, Time, Description, Location, Latitude, Longitude, Accident, Personal Injury, Property Damage, Fatal, Commercial License, Hazardous Materials, Commercial Vehicle, Alcohol, Work Zone, Vehicle Type, Year, Make, Model, Color, Violation Type, Contributed to Accident, Driver's Race, Driver's Gender, Driver's State, Driver's License State

Keywords: traffic violations, electronic tickets, county data, privacy protection, dataset management
