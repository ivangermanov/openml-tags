Experiment Data Obtained by Running Random Configurations of Generalized Linear Model Networks Through Machine Learning in R on Various Classification Tasks from OpenML.

This dataset contains experimental results gathered from running randomized configurations of generalized linear model networks through machine learning in R on multiple classification tasks sourced from OpenML.

The dataset features include the task identification, dataset name, learner type, performance metrics such as mean misclassification error and log loss, training time, prediction time, and the number of imputed selected cross-validation parameters.
