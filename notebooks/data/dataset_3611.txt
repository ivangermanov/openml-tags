Ulaanbaatar Weather Dataset 

Tags: Computer Systems, Machine Learning 

You can access a detailed weather data set of Ulaanbaatar, the capital city of Mongolia. This dataset includes timestamps in Coordinated Universal Time (UTC) and timely basis data of various weather-related features.

Since there is a significant demand for using weather data, but not enough free materials are available, this dataset is being shared to fill this gap.

The dataset description is as follows: The data includes easy-to-access information on temperature, wind, and other weather conditions, which can be found on www.wunderground.com. This comprehensive dataset encompasses several features, including expiration time, valid time, day indicator, temperature, weather icon, weather phrase, dew point, heat index, relative humidity, pressure, visibility, wind chill, wind direction, wind speed, ultraviolet description, feels-like temperature, ultraviolet index, and cloud cover, along with the corresponding dates.
