The AutoML Challenge dataset is a regression task that was part of the AutoML Challenge Series. The test and validation sets can be obtained on the ChaLearn website. This dataset consists of a large number of features, which are not human-readable. 

The dataset was originally used in the AutoML Challenge 2014 and is available for download on the ChaLearn website. The dataset is part of the AutoML Challenge Series, which is a series of challenges focused on automated machine learning.
