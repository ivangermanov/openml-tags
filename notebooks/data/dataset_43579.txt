The Crimes in Los Angeles from 2010 dataset reflects incidents of crime in the City of Los Angeles, dating back to 2010. This data is transcribed from original crime reports that are typed on paper, and therefore, there may be some inaccuracies within the data. Some location fields with missing data are noted as unknown locations. Address fields are only provided to the nearest hundred block in order to maintain privacy. This data is as accurate as the data in the database.

This dataset was taken from data.gov and includes information about the date and time when the crime took place, the area where it happened, the type of crime committed, and the weapon used. The dataset provides a comprehensive view of crimes committed in Los Angeles, including details about the victims, premises, and status of each case.

Features of this dataset include the date reported, date occurred, time occurred, area ID, area name, reporting district, crime code description, victim age, victim sex, victim descent, premise code description, weapon used description, status code description, and address.
