This dataset reflects incidents of crime in the City of Los Angeles dating back to the year two thousand and ten. This data is transcribed from original crime reports that are typed on paper and therefore there may be some inaccuracies within the data. Some location fields with missing data are noted as unknown. Address fields are only provided to the nearest hundred block in order to maintain privacy. This data is as accurate as the data in the database. This data was taken from data.gov.

The dataset includes information on the date and time when the crime took place, the area where it happened, the type of crime, and the weapon used. The dataset provides a comprehensive overview of crime incidents in Los Angeles, including the reporting district, crime code, and crime code description. Additionally, the dataset includes information on the victim, such as age, sex, and descent, as well as the premise code and premise description.

The dataset also includes information on the weapon used, including the weapon used code and weapon description. Furthermore, the dataset provides information on the status of the crime, including the status code and status description. The dataset also includes additional crime codes, which provide further information on the incident.

The address of the crime is also included, as well as the cross street and location. This information can be used to identify patterns and trends in crime incidents in Los Angeles.
