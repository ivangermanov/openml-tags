LED Concept Drift Study Dataset 

This dataset is characterized by the presence of concept drift, which means that the underlying distribution of the data changes over time. It is part of Study Sixteen and comprises a large number of instances. 

The dataset features a range of attributes including twenty-four distinct characteristics. These attributes are used to describe each instance in the dataset.

The primary focus of this dataset is on the class attribute, which is the target variable that needs to be predicted.
