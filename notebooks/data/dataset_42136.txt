This dataset, titled LA Crimes, is tagged with relevant information and reflects incidents of crime in the City of Los Angeles dating back to the year two thousand ten. The data is transcribed from original crime reports that are manually typed on paper, which may lead to some inaccuracies within the dataset. To maintain individual privacy, address fields are only provided to the nearest hundred block.

The accuracy of this data is directly correlated to the accuracy of the data stored in the database. Please note any questions or concerns regarding the dataset in the comments section.

Features of this dataset include:

Date Reported
Date Occurred
Time Occurred
Area Identification Number
Area Name
Reporting District
Crime Code Description
Modus Operandi Codes
Victim Age
Victim Sex
Victim Descent
Premise Code
Premise Description
Weapon Used Code
Weapon Description
Status Code
Status Description
Address
Cross Street
Location
