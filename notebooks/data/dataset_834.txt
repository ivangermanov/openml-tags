Tags: Chemistry, Life Science, Mythbusting, Study 

This binarized version of the original dataset converts the numeric target feature into a two-class nominal target feature by computing the mean and classifying all instances with a lower target value as Positive and all others as Negative.

The dataset consists of various chemical compounds, each described by multiple features, which are essential for understanding their properties and behavior.

Features: Oxygen One, Oxygen Two,..., Oxygen Ninety Nine, Oxygen One Hundred
