California Dataset

This dataset is used in the Tabular Data Benchmark and has been transformed in the same way. It belongs to the Regression on Numerical Features benchmark. The original source of this dataset is available online.

The dataset features include Median Income, House Age, Average Rooms, Average Bedrooms, Population, Average Occupancy, Latitude, Longitude, and Price.

Please give credit to the original source if you use this dataset.
