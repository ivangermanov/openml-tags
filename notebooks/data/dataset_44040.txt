Analcatdata Supreme

This dataset is used in the tabular data benchmark and has been transformed in the same way. It belongs to the "regression on categorical and numerical features" benchmark.

The original dataset is a collection of data sets used in the book "Analyzing Categorical Data". The submission consists of a zip file containing two versions of each of the data sets, plus a README file. Each data set is given in comma-delimited ASCII form and Microsoft Excel form.

Please note that these data sets may be used freely for scientific, educational, and/or non-commercial purposes, provided suitable acknowledgment is given by citing the reference.

Further details concerning the book, including information on statistical software, are available at the website.

Information about the dataset: 
Class Type is numeric, and there is no specific Class Index.

The features of this dataset include Actions Taken, Liberal, Unconstitutional, Precedent Alteration, Unanimous, Year of Decision, Lower Court Disagreement, and Log Exposure.

Keywords: categorical data, regression, numerical features, data benchmark, statistical software.
