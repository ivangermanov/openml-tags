Analcat Data Michigan Acc Dataset:

Tags: Chemistry, Life Science, Myth Busting, Study

This dataset is a binarized version of the original dataset. It involves converting the numeric target feature into a two-class nominal target feature by calculating the mean and categorizing all instances with a lower target value as Positive and all others as Negative.

The features included in this dataset are Time Index, Season, Month, and Unemployment Rate, which are used to classify instances into Positive or Negative categories.
