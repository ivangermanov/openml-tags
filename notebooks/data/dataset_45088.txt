Diffuse Large B-Cell Lymphoma Dataset Tags: Microarray

This dataset is a collection of gene expression profiles from patients with Diffuse Large B-Cell Lymphoma, a type of cancer that affects the immune system. The dataset was created to predict the outcome of patients with this disease using gene expression profiling and supervised machine learning.

The dataset features a large number of attributes, including various gene expression levels, which are used to train machine learning models to predict patient outcomes.
