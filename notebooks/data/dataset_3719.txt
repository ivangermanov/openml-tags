The One Hundred Fifty Thousand Lyrics Labeled with Spotify Valence Tags dataset is categorized under Computer Systems and Machine Learning Context. This dataset was created to perform a supervised natural language processing sentiment analysis task utilizing the Spotify valence audio feature, which measures the overall positivity of a song.

The preparation process of the dataset is thoroughly explained in the accompanying notebook. 

This dataset draws inspiration from analyzing lyrics in relation to various song audio features.

Features include Artist, Sequence, Song, and Label.

Keywords: Sentiment Analysis, Natural Language Processing, Spotify Valence, Audio Features, Lyrics Analysis, Music Dataset
