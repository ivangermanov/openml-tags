YouTube Spam Collection LMFAO Dataset

This dataset is a public set of comments collected for spam research. It has five datasets composed of real messages extracted from five videos that were among the most viewed during the collection period. This dataset only contains information about LMFAO.

The dataset consists of spam entries and ham entries, leading to a grand total of samples. 

Attribute Information

The collection is composed of one CSV file per dataset, where each line has the following attributes: Comment ID, Author, Date, Content, and Class.
