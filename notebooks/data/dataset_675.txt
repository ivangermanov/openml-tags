This dataset contains twenty two data sets from the book Visualizing Data published by Hobart Press. The dump was created using the data.dump function and can be read back into S by using the data.restore function. Each S dataset has been given a unique name which corresponds to the name of the dataset used in the book. To find a detailed description of each dataset, readers can refer to the index of the book under the entry "Data" followed by the name of the dataset. For instance, one of the datasets is named Barley. Readers can find more information about this dataset by looking up the index under the entry "Data, Barley".
