Amazon Commerce Reviews Dataset Subsampling with Seed Zero

This dataset is a subsampling of the Amazon Commerce Reviews dataset with a seed of zero. The subsampling was done using a specific set of arguments, including the number of rows, number of columns, and number of classes.

The subsampling process involved uniformly sampling classes from the original dataset, selecting a subset of columns, and then stratifying the data accordingly. The goal was to create a smaller dataset that still maintains the overall characteristics of the original dataset.

The resulting dataset has a specific set of features, including product reviews and ratings. The features are a combination of categorical and numerical variables, which can be used for training machine learning models.

The dataset is useful for tasks such as sentiment analysis, product recommendation, and customer review analysis.

Keywords: Amazon Commerce Reviews, dataset subsampling, machine learning, sentiment analysis, product recommendation.
