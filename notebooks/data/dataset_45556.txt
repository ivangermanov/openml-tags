This dataset is a subset of the KDD Cup 2012 Track Two data, created for the paper Gated Additive Tree Ensemble for Tabular Classification and Regression. We retrieved the data from Dropbox. Note that please read the Kaggle dataset description carefully before using this dataset. This dataset mostly contains IDs that should be looked up in other files that are not on OpenML and were not used as part of the benchmark in the paper mentioned above. The dataset features include impression, URL hash, ad identifier, advertiser identifier, depth, position, query identifier, keyword identifier, title identifier, description identifier, and user identifier, as well as a target variable.
