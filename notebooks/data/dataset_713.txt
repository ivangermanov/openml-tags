Fri C One Five Hundred Five Dataset 

Tags: Binarized Regression Problem, Chemistry, Life Science, Mythbusting One, Study One, Study Fifteen, Study Twenty, Study Forty-One, Study Seven, Study Eighty-Eight 

This dataset is a binarized version of the original data set. The numeric target feature has been converted into a two-class nominal target feature by calculating the mean and categorizing all instances with a lower target value as Positive and all others as Negative. The features included in this dataset are oz one, oz two, oz three, oz four, and oz five, along with the binary class feature.
