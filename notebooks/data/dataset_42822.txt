Name: NASA PHM Dataset
Tags: Features of the Dataset

This dataset contains a wide range of features, including twenty-one sensors that collect various types of data. These sensors are designed to monitor and track different aspects of the system, providing valuable insights and information. The features of this dataset are comprised of sensor one, sensor two, sensor three, and so on, all the way up to sensor twenty-one. Additionally, the dataset includes a class feature that categorizes the data into different groups.
