**Internet Firewall Data Set**

This data set was collected from the internet traffic records on a university's firewall. There are twelve features in total. The Action feature is used as a class, and there are four classes in total, namely Allow, Action, Drop, and Reset Both classes.

**Attribute Information**

The data set includes the following attributes: Source Port, Destination Port, NAT Source Port, NAT Destination Port, Action, Bytes, Bytes Sent, Bytes Received, Packets, Elapsed Time in seconds, Packets Sent, and Packets Received.

**Scraped Data**

The data set information is provided, along with additional information about the data set. There are twelve features in total, and the Action feature is used as a class. There are four classes in total, namely Allow, Action, Drop, and Reset Both classes. The data set does not have any missing values.
