Tags: Machine Learning, Text And Literature, Bleichenbacher Timing Attack

Attribute Descriptions:

This dataset contains attributes describing various aspects of TCP packets and TLS alerts.

CCS Zero attributes describe the first Change Cipher Spec TCP acknowledgement packet.

TCP source port, destination port, source or destination port, TCP stream index, TCP segment length, sequence number, next sequence number, acknowledgement number, header length, reserved flag, nonce flag, congestion window reduced flag, ECN echo flag, urgent flag, acknowledgement flag, push flag, reset flag, syn flag, fin flag, window size value, calculated window size, window size scale factor, checksum status, urgent pointer, NOP option, option kind, option length, timestamp value, time delta, and message order are all included in the CCS Zero attributes.

TLS Zero attributes describe the first TLS alert packet.

TCP source port, destination port, source or destination port, TCP stream index, TCP segment length, sequence number, next sequence number, acknowledgement number, header length, reserved flag, nonce flag, congestion window reduced flag, ECN echo flag, urgent flag, acknowledgement flag, push flag, reset flag, syn flag, fin flag, window size value, calculated window size, window size scale factor, checksum status, urgent pointer, NOP option, option kind, option length, timestamp value, time delta, TLS record content type, TLS record length, TLS alert message level, TLS alert message description, and message order are all included in the TLS Zero attributes.

Disconnect Zero attributes describe the first TCP disconnect packet.

TCP source port, destination port, source or destination port, TCP stream index, TCP segment length, sequence number, next sequence number, acknowledgement number, header length, reserved flag, nonce flag, congestion window reduced flag, ECN echo flag, urgent flag, acknowledgement flag, push flag, reset flag, syn flag, fin flag, window size value, calculated window size, window size scale factor, checksum status, urgent pointer, NOP option, option kind, option length, timestamp value, time delta, and message order are all included in the Disconnect Zero attributes.

Disconnect One attributes describe the second TCP disconnect packet.

TCP source port, destination port, source or destination port, TCP stream index, TCP segment length, sequence number, next sequence number, acknowledgement number, header length, reserved flag, nonce flag, congestion window reduced flag, ECN echo flag, urgent flag, acknowledgement flag, push flag, reset flag, syn flag, fin flag, window size value, calculated window size, window size scale factor, checksum status, urgent pointer, NOP option, option kind, option length, timestamp value, time delta, and message order are all included in the Disconnect One attributes.

Features include label, CCS Zero TCP source port, CCS Zero TCP destination port, and many others.

Fold ID and vulnerable classes are also included.
