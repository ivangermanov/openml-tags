The Andrew Ng Machine Learning Tweets Dataset

This dataset consists of tweets regarding Machine Learning, focusing on Earth Science and Machine Learning context. It provides a unique opportunity for those interested in Natural Language Processing.

The dataset contains eight columns, including ID, Username, Name, Tweet, Conversation ID, Time Zone, Time, and Polarity. The Polarity column is particularly noteworthy, as it represents the sentiment analyzed column, categorizing tweets into different sentiments.

This dataset is ideal for working on Sentiment Analysis and NLP projects. It offers a wealth of information for those looking to explore the intersection of machine learning and earth science.

Keywords: Andrew Ng, Machine Learning, Tweets, Earth Science, Natural Language Processing, Sentiment Analysis, NLP
