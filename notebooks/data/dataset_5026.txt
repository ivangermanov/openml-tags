**Link Bayesian Network**

This Bayesian network repository reference is available online.

The Link Bayesian Network has seven hundred twenty-four nodes and one thousand one hundred twenty-five arcs. It also has fourteen thousand two hundred eleven parameters. The average Markov blanket size is approximately four point eight. The average degree is approximately three point one one.

This network was created for linkage analysis in large pedigrees with many loops.

Features include variables related to genetic markers, pedigree structures, and other factors influencing linkage analysis.

Keywords: Bayesian network, linkage analysis, pedigree, genetic markers, Markov blanket.
