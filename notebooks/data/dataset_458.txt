analcatdata Authorship Dataset

This dataset is part of the collection of datasets used in the book "Analyzing Categorical Data". It is comprised of eighty-four datasets, each presented in two formats: comma-delimited ASCII and Microsoft Excel.

The submission includes a zip file containing the datasets, along with this README file. Users are permitted to utilize these datasets for scientific, educational, and/or non-commercial purposes, provided they acknowledge the source by citing the aforementioned reference.

Further information regarding the book, including details on statistical software and sample code, can be accessed at the website.

The dataset characteristics are as follows:

Class type: nominal
Class index: last

Note: Quotes, single quotes, and backslashes were removed, while blanks were replaced with spaces.

Features: Book ID
