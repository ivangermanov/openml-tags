The dataset provides qualitative and quantitative data about thirty-one projects completed in a consumer electronics company, with one row per project. It includes a mixture of qualitative attributes measured on a five-point ranked scale (Very Low, Low, Medium, High, Very High) and quantitative attributes with specified scales.

The project data incorporates qualitative factors for improved software defect prediction. The dataset includes various attributes related to software development, testing, and project management.

Keywords: software engineering, defect prediction, PROMISE dataset, qualitative factors, software development, testing, project management.
