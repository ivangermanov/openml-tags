Internet Advertisements Tags: OpenML-CC18, study_135, study_99

Description:
This dataset represents a set of possible advertisements on Internet pages. The task is to predict whether an image is an advertisement or not.

Sources:
This dataset was created and donated by the University of California, Irvine.

Dataset Information:
The features encode the geometry of the image, phrases occurring in the URL, the image's URL and alt text, the anchor text, and words occurring near the anchor text.

Attributes Information:
There are three continuous attributes, and the others are binary. One or more of the three continuous features are missing in some instances, which should be interpreted as "unknown".

Relevant Papers:
Learning to remove Internet advertisements.

Features:
Image height, image width, aspect ratio, and phrases occurring in the URL, such as local, images, buttons, likes books, and more.

Keywords: Internet Advertisements, Image Classification, URL Phrases, Geometry Features
