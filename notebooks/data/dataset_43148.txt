The Stack Overflow Polarity Test is a gold standard dataset consisting of approximately four thousand four hundred questions, answers, and comments from Stack Overflow, manually annotated for sentiment polarity. This dataset has been utilized for developing the Emotional Manipulation Toolkit for detecting sentiment polarity from technical text. The dataset features include a unique identifier, sentiment polarity, and the text itself.

Keywords: sentiment analysis, polarity detection, technical text, emotional manipulation, gold standard dataset.
