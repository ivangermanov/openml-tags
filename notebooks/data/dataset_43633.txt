NYC Housing Data Dataset

Tags: Computer Systems, Machine Learning

This dataset has been created for the purpose of predicting future New York housing prices based on a comprehensive seventeen-year record of historical data.

Content:

Please refer to the detailed descriptions provided under each column for further information.

Acknowledgments:

The data used in this dataset has been sourced from the New York City Department of Finance's open-source database.

Inspiration:

This dataset was inspired by a school project, specifically focusing on the development of a data definition language.

Features:

Borough, Neighborhood, Building Class Category, Address, Zip Code, Land Square Feet, Gross Square Feet, Year Built, Sale Price, and Sale Date.

Keywords: New York Housing, Predictive Modeling, Historical Data, Real Estate, Computer Systems, Machine Learning
