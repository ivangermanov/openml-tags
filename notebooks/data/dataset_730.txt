The Fri C One Hundred Twenty Five dataset is categorized under Chemistry, Life Science, and Mythbusting. This binarized version of the original dataset converts the numeric target feature into a two-class nominal target feature. This conversion is achieved by calculating the mean and subsequently classifying all instances with a lower target value as Positive and all others as Negative.

Features of this dataset include various chemical properties, although their specific identities are not readily apparent.
