Emissions by Cars
Tags: Context

This dataset contains various information about a set of cars that were manufactured with a set of factory parameters like cylinder size, number of cylinders, fuel consumption, and carbon dioxide emissions. The content feature set includes the car model, make, vehicle type, engine size, transmission, fuel consumption, and other relevant details. The target set is the carbon dioxide emissions, which is a critical aspect of understanding the environmental impact of vehicles.

The inspiration behind this dataset was to learn and help understand the concept behind data science's basics, particularly in the context of analyzing the relationship between various car features and their carbon footprint.

Features: Car Model, Make, Vehicle Class, Engine Size, Number of Cylinders, Transmission, Fuel Type, Fuel Consumption, and Carbon Dioxide Emissions.

Keywords: Carbon Dioxide Emissions, Fuel Consumption, Engine Size, Vehicle Features, Data Science.
