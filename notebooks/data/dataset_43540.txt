Emissions by Cars 
Tags: Machine Learning, Mathematics 

This dataset contains various information about a set of cars that were manufactured with a specific set of factory parameters, such as cylinder size, number of cylinders, fuel consumption, and carbon dioxide emissions. The content feature set includes details about the car model, make, vehicle type, engine size, transmission, and fuel consumption. The target variable of interest is the carbon dioxide emission level.

The inspiration behind this dataset was to gain a deeper understanding of the fundamental concepts underlying data science. By analyzing this dataset, one can explore the relationships between different car attributes and their impact on carbon dioxide emissions.

Features: Model, Make, Vehicle Class, Engine Size, Cylinders, Transmission, Fuel Type, Fuel Consumption, Carbon Dioxide Emissions

Keywords: Machine Learning, Carbon Dioxide Emissions, Fuel Consumption, Car Attributes, Data Science
