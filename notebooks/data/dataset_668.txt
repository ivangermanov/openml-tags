The Census 1980 dataset is a collection of data from the book Data Analysis: An Introduction. This dataset is classified under the categories of Chemistry, Life Science, and StatLib.

A shared archive of data has been submitted, which contains information about various states. The dataset does not have a specific class index.

The features of this dataset include State, Percentage of Population Over Sixty Five, Median Age, Per Capita Income, Percentage of College Educated Population, and Percentage of High School Educated Population.
