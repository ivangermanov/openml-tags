This is one of forty-one drug design datasets. The datasets with eleven hundred forty-three features are formed using Adriana Code software. The molecules and outputs are taken from the original studies.

The Carbolenes dataset is formed using data from the Journal of Medicinal Chemistry. The MTP Two dataset is formed using data from the Journal of Chemical Information and Computer Science.

The Chang, Cristalli, Depreux, Doherty, Garrat Two, Garrat, Heyl, Krystek, Lewis, Penning, Rosowsky, Siddiqi, Stevenson, Strupcz, Svensson, Thompson, Tsutumi, Uejling, Yokoyama One, and Yokoyama Two datasets are formed using data from the Journal of Medicinal Chemistry.

The MTP dataset is formed using data from the Journal of Chemical Information and Modeling. The Benzo Thirty-Two dataset is formed using data from the European Journal of Medicinal Chemistry.

The Phenethyl One dataset is formed using data from the book QSAR: Hansch Analysis and Related Approaches. The PAH dataset is formed using data from the journal Chemometrics and Intelligent Laboratory Systems.

The PDGF-R dataset is formed using data from the Journal of Chemical Information and Computer Science. The Phen dataset is formed using data from the Journal of Medicinal Chemistry.

The Topo Two One and Y Prop Four One datasets are formed using data from the Journal of Chemical Information and Computer Science. The QSAR BR One, QSAR BR Two, QSAR Tox, QSBR RW One, QSBR Y Two, QSBR Alks, QSFR DHLa, QSFSR One, QSFSR Two, and QSPrCmpX datasets are formed using various sources including books and journals.

The Selwood dataset is formed using data from the Journal of Medicinal Chemistry.

Keywords: drug design, machine learning, medicine, QSAR, chemical information, molecular descriptors.
