This dataset is a playground for fundamental and technical analysis, serving as a basic tutorial for time-series data analysis. 

The dataset consists of the following files: LT Finance Holdings Ltd Stock Price from February to February, containing thirteen columns.

The prices were fetched from the BSE India databases.

This dataset can be used to try out various things, such as one day ahead prediction using rolling linear regression, ARIMA, SARIMA, LSTM, momentum/mean-reversion strategies.

The dataset features include Date, Open Price, High Price, Low Price, Close Price, Weighted Average Price, Number of Shares, Number of Trades, Total Turnover, Deliverable Quantity, and Percentage of Deliverable Quantity to Traded Quantity.

Keywords: stock price, time-series analysis, technical analysis, fundamental analysis, prediction, regression, momentum strategy.
