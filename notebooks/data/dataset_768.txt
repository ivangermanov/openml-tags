Fri C Three Hundred One Dataset 

Tags: Chemistry, Life Science, Myth Busting One, Study One, Study Fifteen, Study Twenty, Study Forty-One, Study Seven, Study Eighty-Eight 

This dataset is a binarized version of the original dataset. It converts the numeric target feature to a two-class nominal target feature by computing the mean and classifying all instances with a lower target value as Positive and all others as Negative. 

Features include twenty-five variables related to ozone measurements.
