Bitcoin Stock Data Tags: Context

This dataset contains the value of the Bitcoin stock from a certain point in time till the present day. The content of this dataset is very simple to both explore and understand, as the columns are themselves descriptive in nature.

Acknowledgements
The source of this dataset is Yahoo Finance.

Inspiration
Just explore the dataset and practice your skills using this really simple real-world dataset.

License
This dataset is licensed under CC0, which means it is in the public domain.

Features
The dataset includes the following features: Date, Opening Price, Highest Price, Lowest Price, Closing Price, and Volume of Trades.
