Disaster Tweets 

Tags: Computer Systems, Machine Learning 

Context 
The file contains tweets associated with disaster keywords such as crash, quarantine, and bushfires, as well as the location and keyword itself. The data structure was inherited from Disasters on Social Media. The tweets were collected on a specific date. Some of the topics people were tweeting about included the eruption of Taal Volcano in the Philippines, the Coronavirus outbreak, and the devastating bushfires in Australia, as well as the tragic event of the Iranian downing of an airplane.

Disclaimer 
Please note that the dataset contains text that may be considered offensive or inappropriate.

Inspiration 
The intention behind creating this dataset was to enrich the already available data on this topic by collecting and manually classifying new tweets. This dataset builds upon the initial source, Disasters on Social Media, which was used in the Real or Not? NLP with Disaster Tweets competition.

Features 
The dataset includes the tweet identifier, the relevant keyword, the location mentioned in the tweet, the actual text of the tweet, and the target label.

Keywords: disaster, tweets, computer systems, machine learning, coronavirus, bushfires, volcano eruption
