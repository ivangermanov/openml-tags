This dataset contains QSAR data showing activity values of several compounds on drug target Dual-Specificity Tyrosine-Phosphorylation Regulated Kinase One A. It has multiple rows and features, including identifiers and class features. The features represent fingerprints which were generated from SMILES strings. They were obtained using the Pipeline Pilot program.

The dataset is part of the MetaQSAR project and contains information about the target protein, including its synonyms, species group, protein target classification, and description. The target protein is a single protein and is classified under the enzyme/kinase/protein kinase group.

The dataset also includes information about the expression atlas, component, function, process, and other related aspects of the target protein. Additionally, it provides links to various databases and resources, such as UniProt, InterPro, Pfam, and PharmGKB.

Keywords: QSAR, MetaQSAR, Dual-Specificity Tyrosine-Phosphorylation Regulated Kinase One A, drug target, protein kinase, enzyme, fingerprint, SMILES string.
