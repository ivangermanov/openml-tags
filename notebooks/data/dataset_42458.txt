Experiment Data Obtained by Running Random Configurations of Ranger through MLR on Various Classification Tasks from OpenML.

This dataset contains experiment data obtained by running random configurations of the Ranger algorithm through the MLR framework on a diverse range of classification tasks from OpenML. 

The dataset features include the task identification, dataset name, learner used, performance metrics such as mean misclassification error and log loss, training and prediction times, and various hyperparameters of the Ranger algorithm, including the number of imputed values selected, number of trees, replacement strategy, sample fraction, respect for unordered factors, minimum node size, split rule, number of random splits, and the number of features to consider at each split.

Tags: Experiment Data, Classification Tasks, OpenML.

Keywords: Machine Learning, Classification, Hyperparameter Tuning, Random Forest, Performance Metrics.
