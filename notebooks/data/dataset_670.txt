The chscase Census Five dataset is a collection of data sets used in the book "A Casebook for a First Course in Statistics and Data Analysis". This dataset was submitted and consists of thirty-eight files, including this Read Me file. Each file represents a data set analyzed in the book, with file names corresponding to those used in the publication.

The data files are written in plain ASCII character text, with missing values represented by "M". For more information about the data sets and the book, please visit the gopher address swis.stern.nyu.edu, where the information is filed under Academic Departments and Research Centers, then Statistics and Operations Research, then Publications, and finally A Casebook for a First Course in Statistics and Data Analysis.

Alternatively, access the information through the World Wide Web using a web browser, starting from the URL address http://www.stern.nyu.edu/SOR/Casebook. Please note that these datasets may be used freely for scientific, educational, and/or non-commercial purposes, provided suitable acknowledgment is given by citing the reference.
