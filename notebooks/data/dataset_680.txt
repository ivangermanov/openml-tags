A collection of data sets used in the book "A Casebook for a First Course in Statistics and Data Analysis," published by John Wiley and Sons, New York. The data sets are available in plain ASCII text format, with missing values represented by "M" across all files. The dataset names correspond to those used in the book.

More information about the data sets and the book can be obtained via gopher at the address swis.stern.nyu.edu, filed under Academic Departments & Research Centers, Statistics and Operations Research, Publications, A Casebook for a First Course in Statistics and Data Analysis, Welcome! It can also be accessed from the World Wide Web using a browser, starting from the URL address provided.

These datasets may be used freely for scientific, educational, and/or non-commercial purposes, provided suitable acknowledgment is given by citing the reference mentioned above.

The specific file discussed is funds.dat. Attribute names were generated automatically as there was no information in the data itself. The dataset is of numeric class type with no specific class index.
