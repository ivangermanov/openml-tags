QSAR-TID-234 Dataset

The QSAR-TID-234 dataset contains QSAR data showing activity values of several compounds on drug target TID: 234. This dataset has multiple rows and features, including identifiers and a class feature.

The features in this dataset represent fingerprints generated from SMILES strings using the Pipeline Pilot program, Dassault Syst√®mes BIOVIA.
