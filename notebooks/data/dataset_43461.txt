Tamil Nadu Crop Production

Tags: Context Agriculture Crop Production in Tamil Nadu, India

This dataset describes the agriculture crop production in Tamil Nadu, India. It is sourced from https://data.gov.in/ and is fully licensed.

Acknowledgements
This dataset was created during an internship at TactLabs.

The target variable is Production, although it is incomplete. The features of this dataset include State Name, District Name, Crop Year, Season, Crop, Area, and Production.
