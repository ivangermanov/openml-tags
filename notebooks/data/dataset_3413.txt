Data Shows the Downlink Goodput for Unicast and Multicast Transmissions with Different Group Sizes and Network Loads on an IEEE 802.11ac Network Simulated on NS-3. 

Different IEEE 802.11aa Group Address Translation Services are used. The features include Level, Number of Multicast Stations, Number of Unicast Stations, Index, Average Data Rate Downlink, Average Data Rate Uplink, Delay Downlink Multicast, Delay Downlink Unicast, Delay Uplink, Goodput Downlink Multicast, Goodput Downlink Unicast, Goodput Uplink, Injected Multicast, Injected Unicast Down, Injected Unicast Up, Multicast Mode, Occupancy, Occupancy Down, Occupancy Up, Retries, Retransmitted Bytes Unicast Downlink, Retransmitted Bytes Unicast Uplink, Retransmission Share Downlink, Retransmission Share Uplink, Received Bytes Downlink Multicast, Received Bytes Downlink Unicast, Received Bytes Uplink, Throughput Multicast Downlink, Throughput Unicast Downlink, Throughput Uplink, Time, Transmitted Bytes Multicast Downlink, Transmitted Bytes Unicast Downlink, and Transmitted Bytes Uplink.
