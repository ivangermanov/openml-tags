French Open Women 2013

Tags: Tennis Major Tournament Match Statistics

This dataset is part of a collection of eight files containing match statistics for both women and men at the four major tennis tournaments of the year 2013. Each file has forty-two columns and a minimum of seventy-six rows. This file contains the statistics for the French Open Women's Edition for 2013.

Attribute Information

The dataset contains information about the players, including their names, and the results of the matches, including the winner and the number of games won. It also includes statistics about the players' performance, such as their first serve percentage, second serve percentage, aces, double faults, winners, unforced errors, break points created and won, net points attempted and won, and total points won. Additionally, the dataset includes information about the sets played, including the result of each set.

The dataset has missing values.
