**Meta-Album Subcellular Human Protein Dataset (Extended)**

This dataset is a subset of the Subcellular dataset in the Protein Atlas project. The original dataset stems from the Human Protein Atlas Image Classification Kaggle competition. We performed two modifications to create this extended version: first, we turned the dataset into a multi-class dataset by dropping all images belonging to more than a single class and also those images that belong to classes with less than forty members; second, we converted the remaining images into RGB by dropping the yellow channel, a common practice in the competition.
