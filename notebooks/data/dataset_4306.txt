**Meta-Album Pan-Nuke Dataset (Extended)**
The Pan-Nuke dataset is a semi-automatically generated segmentation and classification task of nuclei. The dataset contains images of nineteen different tissue types. For the Meta-Album meta-dataset, even though this dataset was designed as a segmentation task, we were able to transform it into a tissue classification task since we had the tissue type for each sample in the dataset. We also resized the images to one hundred twenty-eight by one hundred twenty-eight pixels and applied stain normalization to avoid bias and remove some spurious features.
