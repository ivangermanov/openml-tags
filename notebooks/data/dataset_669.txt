Tags: StatLib

A collection of data sets used in the book "A Casebook for a First Course in Statistics and Data Analysis," published by John Wiley and Sons, New York. The data sets are available in plain ASCII text format, with missing values represented by "M" in all files. Each file corresponds to a data set analyzed in the book, with file names matching those used in the publication.

More information about the data sets and the book can be obtained via gopher or the World Wide Web. The data is accessible through the Statistics and Operations Research department's publications section at the provided web address.

These datasets may be used freely for scientific, educational, and non commercial purposes, provided suitable acknowledgment is given by citing the reference to the book.

The specific file "adopt.dat" is included in this collection. Attribute names for this dataset were generated automatically due to lack of information in the data itself. The dataset is classified as numeric with no specific class index.
