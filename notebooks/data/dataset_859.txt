Anal Cat Data G Violence Dataset 

Tags: Chemistry, Life Science, Myth Busting 

This dataset is a binarized version of the original data set. The numeric target feature has been converted into a two-class nominal target feature. This was achieved by computing the mean and then classifying all instances with a lower target value as Positive and all others as Negative.

The dataset includes the following features: Title, Year, Length, Time of Violence, Number of Injuries, Fatal Injuries, Neutral or Good Injuries, Neutral or Good Fatalities.

Keywords: violence, injuries, fatalities, chemistry, life science, myth busting
