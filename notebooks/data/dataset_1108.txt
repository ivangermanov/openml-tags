The Four-Class Dataset 

Tags: Chemistry, Life Science 

This dataset originates from the LIBSVM data repository. It was obtained from an original source, although the exact date of acquisition is unknown.

To cite this dataset, please reference the following publication: Building Projectable Classifiers of Arbitrary Complexity, which was presented at the International Conference on Pattern Recognition.

Preprocessing involved transforming the data into a two-class format.

The dataset consists of the following features: Attribute One, Attribute Two, and Class.

Keywords: LIBSVM, Pattern Recognition, Classification, Data Repository, Dataset.
