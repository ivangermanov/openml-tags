This dataset has been scraped off Goodreads to obtain landmark information about the most exceptional literary works of the Nineteenth Century. 

Feature Description 
The Book Title represents the title of the book. 
A Brief Summary provides a concise description about the book. 
The Overall Rating represents the rating given by Goodreads users. 
The Book Cover Image Link is the link to the book cover.

Features: Book Title, Brief Summary, Overall Rating, Book Cover Image Link

Keywords: Nineteenth Century, Literary Works, Goodreads, Book Title, Brief Summary, Overall Rating, Book Cover Image Link
