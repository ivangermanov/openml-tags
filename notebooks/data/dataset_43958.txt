The KDD IPUMS LA Ninety Seven Small dataset is categorized under Machine Learning and Statistics. It is featured in the tabular data benchmark, specifically in the classification on numerical features category.

This dataset has been transformed in the same manner as the original dataset, which was modified to create a binarized version of the original data. The multi-class target feature has been converted into a two-class nominal target feature, where the majority class is labeled as Positive and all other classes are labeled as Negative.

The dataset contains various features, including Value, Rent, Total Income, Mother's Location, Family Size, Number of Children, Elderly Children, Young Children, Number of Siblings, Age, Occupation Score, Socioeconomic Index, Total Income, Wage Income, Business Income, Farm Income, Social Security Income, Welfare Income, Other Income, Poverty Level, and Binary Class.

Keywords: Machine Learning, Statistics, Tabular Data Benchmark, Classification, Numerical Features, KDD IPUMS LA Ninety Seven Small, Binarization, Nominal Target Feature.
