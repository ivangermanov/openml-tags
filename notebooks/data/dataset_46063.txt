Name: Stylized Meta Album Remote Sensing Image Classification Benchmark

Tags: Remote Sensing Dataset (Stylized)

Features: Category, File Name, File Path, Style, Style Name, Style Path, Stylized Path

This dataset, known as the Stylized Meta Album Remote Sensing Image Classification Benchmark, is a collection of remote sensing images that have been stylized for various purposes. The dataset is categorized into different groups, each with its own unique characteristics. The files are stored in a specific path, with each file having a unique name and style. The style of each file is further classified into different names and paths, which are also stored in a stylized path.
