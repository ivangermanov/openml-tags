Andes Bayesian Network Sample 9

This Bayesian network repository reference is a discrete very large network.

The Andes Bayesian Network has two hundred twenty-three nodes and three hundred thirty-eight arcs. It also has one thousand one hundred fifty-seven parameters. The average Markov blanket size is five point six one, and the average degree is three point zero three. The maximum in-degree is six.

This network was used for online student modeling for coached problem solving using Bayesian networks.

Features include Goal, Resolve, Choose, Identify, Try, Bug, Apply, Weight, Slide, Newtons, Find, Force, Maximize, Equal, Define, Axis, Normalize, Displace, Kinematic, Write, Incline, Composite, and many others.

Keywords: Bayesian Network, Machine Learning, Medicine, Online Student Modeling, Coached Problem Solving
