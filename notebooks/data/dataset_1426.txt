**A5A Dataset**

Tags: Chemistry, Life Science

This dataset originates from the LIBSVM data repository. To ensure proper citation, please refer to http://archive.ics.uci.edu/ml/datasets/Adult.

The original Adult data set consists of fourteen features, comprising six continuous and eight categorical attributes. To facilitate processing, continuous features have been discretized into quantiles, with each quantile represented by a binary feature. Additionally, categorical features with multiple categories have been converted into multiple binary features. Detailed information on the conversion process for each feature can be accessed through the following link: http://research.microsoft.com/en-us/um/people/adult.zip.
