The dataset, known as Public Coverage, falls under the categories of Computer Systems and Machine Learning. This particular dataset was recreated by carefully following the instructions outlined in a research paper. The original data used to create this set comes from the American Community Survey Public Use Microdata Sample.

This dataset includes various features, such as Age, Educational Level, Marital Status, Sex, Disability Status, Employment Status, Citizenship, Migration Patterns, Military Service, Ancestry, Nativity, Hearing Impairment, Eye Problems, Remittance Sent, Personal Income, Employment Ratio, State of Residence, Fertility, and Race.

Keywords: Public Coverage, Computer Systems, Machine Learning, American Community Survey, Data Recreation, Research Paper, Demographic Features
