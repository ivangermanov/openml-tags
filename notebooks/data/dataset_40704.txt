Titanic Tags: Derived PMLB Version of the Titanic Dataset

This dataset is a modified version of the original Titanic dataset, which has been condensed to only include three features. For those interested in exploring the complete dataset, a more comprehensive version is available, which can be accessed through the following link: https://www.openml.org/dataset/titanic.

The three features included in this dataset are the passenger's class, age, and sex, as well as the class label. These features provide a concise yet informative snapshot of the passengers on board the ill-fated ship.

Keywords: Titanic, dataset, PMLB, class, age, sex
