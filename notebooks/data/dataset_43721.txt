Hotel Reviews

Tags: Chemistry, Life Science

Data Description

The dataset is obtained from one of the leading travel websites, featuring hotel reviews provided by customers who have had firsthand experiences at these establishments.

Variable Description

The user ID is a unique identifier assigned to each customer who submits a review. The description variable contains the actual text of the review posted by the customer, which may include their opinions, feedback, and ratings of the hotel. The browser used variable indicates the type of web browser utilized by the customer to submit their review, whereas the device used variable specifies the type of device employed, such as a desktop computer, laptop, tablet, or smartphone. Finally, the is response variable serves as the target variable in this dataset.

Features:

* User ID
* Description
* Browser Used 
* Device Used 
* Is Response

Keywords: hotel reviews, customer feedback, travel website, user experience, online ratings.
