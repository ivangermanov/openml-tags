Connect Four Dataset
Tags: Chemistry, Life Science

This dataset originates from the LIBSVM data repository. The preprocessing step involved using binary encoding for each feature, resulting in a total of one hundred twenty-six features.

The dataset contains various attributes, including att one through forty-two, which have been encoded into a binary format.

To utilize this dataset, please ensure proper citation to the LIBSVM data repository.
