The YouTube Spam Collection Katy Perry dataset is a public set of comments collected for spam research. It has five datasets composed of real messages extracted from five videos that were among the most viewed during the collection period. This dataset only contains information about Katy Perry.

It consists of spam entries and ham entries, leading to a grand total of samples. The collection is composed of one CSV file per dataset, where each line has the following attributes: Comment ID, Author, Date, Content, Class, and Tag.
