QSAR-TID-11003 Tags: MTLQSAR, study_3

This dataset contains QSAR data showing activity values of several compounds on drug target TID: 11003. The dataset has multiple rows and features, including IDs and class feature: Molecule ChEMBL ID and Median PXC50. The features represent FCFP 1024bit Molecular Fingerprints, which were generated from SMILES strings using the Pipeline Pilot program, Dassault Systèmes BIOVIA.

The dataset contains QSAR data from ChEMBL version seventeen, showing activity values of several compounds on drug target TID: 11003. The features were generated from SMILES strings using the Pipeline Pilot program, Dassault Systèmes BIOVIA, and represent FCFP 1024bit Molecular Fingerprints.

The dataset is part of the MetaQSAR project, which aims to provide a comprehensive collection of QSAR datasets for various drug targets. This particular dataset is focused on the Melanocortin receptor three, which is a single protein target that belongs to the family A G protein-coupled receptor.

The Melanocortin receptor three is a membrane receptor that plays a crucial role in various physiological processes, including the regulation of heart rate, blood pressure, and metabolic processes. It is also involved in the regulation of feeding behavior and locomotor rhythm.

The dataset provides a valuable resource for researchers and scientists working on the development of new drugs targeting the Melanocortin receptor three. It can be used to build predictive models and identify potential compounds that can modulate the activity of this receptor.
