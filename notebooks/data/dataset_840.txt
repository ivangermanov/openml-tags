Hypothyroid Dataset

Tags: Chemistry, Life Science, Mythbusting, Study

This dataset is a binarized version of the original data set. The multi-class target feature has been converted to a two-class nominal target feature by relabeling the majority class as Positive and all others as Negative.

The dataset includes various features related to hypothyroidism, including demographic information such as Age and Sex. It also covers medical history, including whether the patient is On Thyroxine, has a Query on Thyroxine, On Antithyroid Medication, Sick, Pregnant, had Thyroid Surgery, received I131 Treatment, and has Queries on Hypothyroid or Hyperthyroid. Additionally, the dataset includes information about medications taken, such as Lithium, and any existing conditions like Goiter, Tumor, Hypopituitary, Psych issues, and Referral Source.

Furthermore, the dataset contains laboratory test results, including measurements for TSH, T3, TT4, T4U, FTI, and TBG.

Keywords: Hypothyroidism, Thyroid, Medications, Laboratory Tests, Demographic Information, Medical History
