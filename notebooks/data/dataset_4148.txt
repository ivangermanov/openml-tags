Dataset used in the Tabular Data Benchmark, transformed in the same way. This dataset belongs to the Regression on Numerical Features benchmark.

Original Description:

The dataset was obtained from an unknown source, and the date of its creation is also unknown. Please cite this dataset when using it.

A binarized version of the original data set has been created. In this version, the multi-class target feature has been converted into a two-class nominal target feature. This conversion involved relabeling the majority class as positive and all other classes as negative.

Features include values related to income, family size, occupation scores, and poverty levels.
