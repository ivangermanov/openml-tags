Traffic Violations Dataset

Tags: Computer Systems, Machine Learning

This dataset contains traffic violation information from all electronic traffic violations issued in the County. To protect privacy, any information that could be used to uniquely identify the vehicle, the vehicle owner, or the officer issuing the violation has been removed.

Source of Original Dataset: https://catalog.data.gov/dataset/traffic-violations

The original dataset was too large and time-consuming to perform basic tasks efficiently. Therefore, it was shuffled and reduced to the first one hundred thousand rows to create this more manageable dataset.

Features:

Date, Time, Description, Location, Latitude, Longitude, Accident, Personal Injury, Property Damage, Fatal, Commercial License, Hazardous Materials, Commercial Vehicle, Alcohol, Work Zone, Vehicle Type, Year, Make, Model, Color, Violation Type, Contributed to Accident, Race, Gender, Driver State, Driver License State

Keywords: Traffic Violations, Electronic Tickets, County Data, Privacy Protection, Data Reduction
