Attribute Descriptions:

The dataset contains attributes related to three types of messages: Change Cipher Spec, TLS Alert, and TCP Disconnect.

The Change Cipher Spec message has several attributes including TCP source port, destination port, port, stream, length, sequence number, next sequence number, acknowledgment number, header length, flags, window size value, window size, window size scale factor, checksum status, urgent pointer, options nop, option kind, option length, timestamp tsval, time delta, and order.

Similarly, the TLS Alert message also has several attributes including TCP source port, destination port, port, stream, length, sequence number, next sequence number, acknowledgment number, header length, flags, window size value, window size, window size scale factor, checksum status, urgent pointer, options nop, option kind, option length, timestamp tsval, time delta, tls record content type, tls record length, tls alert message level, tls alert message description, and order.

Lastly, the TCP Disconnect message also has several attributes including TCP source port, destination port, port, stream, length, sequence number, next sequence number, acknowledgment number, header length, flags, window size value, window size, window size scale factor, checksum status, urgent pointer, options nop, option kind, option length, timestamp tsval, time delta, and order.
