The KDD IPUMS LA 97 dataset has been binarized into a regression problem. This dataset belongs to the categories of Chemistry, Life Science, Mythbusting, and Study datasets. 

The multi-class target feature has been converted into a two-class nominal target feature. This was achieved by re-labeling the majority class as Positive and all other classes as Negative. 

This dataset contains various features including Year, General Quality, General Quality Type, Farm, Ownership Status, Value, Rent, Total Family Income, Number of Families, Number of Couples, Number of Mothers, Number of Fathers, Mother's Location, Stepmother, Mother's Rule, Population Location, Steppopulation, Population Rule, Spouse Location, Spouse Rule, Family Size, Number of Children, Number of Children Less than Five, Family Unit, Elder Child, Youngest Child, Number of Siblings, Relationship, Age, Sex, Race, Marital Status, Birthplace, School, Educational Record, School Type, Employment Status, Labor Force, Occupation, Occupation Score, Socioeconomic Index, Industry, Class of Worker, Weeks Worked, Hours Worked, Year Last Worked, Worked Last Year, Total Income, Wage Income, Business Income, Farm Income, Social Security Income, Welfare Income, Other Income, Poverty Status, Migration Status, Migration Place, Moved In, Veteran Status, and Binary Class.
