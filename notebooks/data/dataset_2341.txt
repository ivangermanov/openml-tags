DBpedia YAGO ARFF

Tags: Geography, Health

This dataset combines DBpedia with the top-most frequent YAGO types for predicting types. It utilizes incoming and outgoing properties as features.

The dataset includes a vast array of features, which are not easily interpretable by humans.

This Hybrid Multiclass Classification dataset is designed for type prediction tasks.
