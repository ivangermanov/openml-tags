Indeed Software Engineer Job Dataset

Tags: Context

This dataset is scraped from the website Indeed.com, and I wanted to scrape this dataset as I was looking for a job and it occurred to me what if I scraped all this data and analyzed it. 

Content 

This dataset contains job profiles from different companies. I scraped ten thousand job profiles, which are represented by ten thousand rows, and each row has six columns. 

Acknowledgements 

I would like to thank Indeed.com for allowing me to scrape their website. 

Inspiration 

Please feel free to analyze this dataset if you want to, and if you find any mistakes in the data, please correct me. Also, if you have any questions about the web scraping process, please do not hesitate to ask. 

Features 

Job Title, Company Name, City, Ratings, Job Summary, Job Post Date
