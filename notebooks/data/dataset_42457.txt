Name: Machine Learning Random Partition Experiment

Tags: Experiment Data Obtained by Running Random Configurations of Decision Trees through Machine Learning on Various Classification Tasks from OpenML

This dataset contains experiment data obtained by running random configurations of decision trees through machine learning on a wide range of classification tasks from OpenML. The dataset features information about the task, including the task ID and dataset name, as well as details about the learner used, such as the performance metrics, training time, and prediction time. Additionally, the dataset includes information about the decision tree parameters, including the number of imputed values selected, complexity parameter, maximum depth, minimum bucket size, and minimum split.

Keywords: Machine Learning, Decision Trees, Classification Tasks, OpenML, Experiment Data
