Porto Seguro Seed One Dataset with Two Thousand Rows, Ten Classes, and One Hundred Columns, Stratified

This dataset is a subsample of the Porto Seguro dataset, generated with a specific set of parameters. The subsampling process involved uniformly selecting classes, columns, and rows from the original dataset.

The dataset has ten classes, which were selected from the original classes using a random sampling method. The selection was done in a way that ensures the classes are represented proportionally to their original frequency.

The dataset also has one hundred columns, which were selected uniformly from the original columns. The selection was done randomly, without any specific criteria.

The dataset has two thousand rows, which were selected using a stratified sampling method. This means that the selection was done in a way that ensures the classes are represented proportionally to their original frequency.

The dataset was generated using a specific seed, which ensures reproducibility of the results.
