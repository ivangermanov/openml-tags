French Open Women 2013
Tags: Images, Life Science

This dataset is part of a collection of eight files containing match statistics for both women and men at the four major tennis tournaments of the year 2013. Each file has forty-two columns and a minimum of seventy-six rows. This file contains the statistics for the French Open Women's Edition for 2013.

Attribute Information:

* Player One: Name of Player One
* Player Two: Name of Player Two
* Result: Result of the match (win or loss) referenced on Player One
* First Serve Percentage for Player One: Real number indicating percentage of successful first serves
* First Serve Won by Player One: Real number indicating percentage of successful first serves won
* Second Serve Percentage for Player One: Real number indicating percentage of successful second serves
* Second Serve Won by Player One: Real number indicating percentage of successful second serves won
* Aces Won by Player One: Integer indicating number of aces won
* Double Faults Committed by Player One: Integer indicating number of double faults committed
* Winners Earned by Player One: Numeric value indicating winners earned
* Unforced Errors Committed by Player One: Numeric value indicating unforced errors committed
* Break Points Created by Player One: Numeric value indicating break points created
* Break Points Won by Player One: Numeric value indicating break points won
* Net Points Attempted by Player One: Numeric value indicating net points attempted
* Net Points Won by Player One: Numeric value indicating net points won
* Total Points Won by Player One: Numeric value indicating total points won
* Set One Result for Player One: Integer indicating set one result
* Set Two Result for Player One: Integer indicating set two result
* Set Three Result for Player One: Integer indicating set three result
* Set Four Result for Player One: Integer indicating set four result
* Set Five Result for Player One: Integer indicating set five result
* Final Number of Games Won by Player One: Integer indicating final number of games won
* First Serve Percentage for Player Two: Real number indicating percentage of successful first serves
* First Serve Won by Player Two: Real number indicating percentage of successful first serves won
* Second Serve Percentage for Player Two: Real number indicating percentage of successful second serves
* Second Serve Won by Player Two: Real number indicating percentage of successful second serves won
* Aces Won by Player Two: Integer indicating number of aces won
* Double Faults Committed by Player Two: Integer indicating number of double faults committed
* Winners Earned by Player Two: Numeric value indicating winners earned
* Unforced Errors Committed by Player Two: Numeric value indicating unforced errors committed
* Break Points Created by Player Two: Numeric value indicating break points created
* Break Points Won by Player Two: Numeric value indicating break points won
* Net Points Attempted by Player Two: Numeric value indicating net points attempted
* Net Points Won by Player Two: Numeric value indicating net points won
* Total Points Won by Player Two: Numeric value indicating total points won
* Set One Result for Player Two: Integer indicating set one result
* Set Two Result for Player Two: Integer indicating set two result
* Set Three Result for Player Two: Integer indicating set three result
* Set Four Result for Player Two: Integer indicating set four result
* Set Five Result for Player Two: Integer indicating set five result
* Final Number of Games Won by Player Two: Integer indicating final number of games won
* Round: Integer indicating round of the tournament at which game is played

Dataset Information:
Has Missing Values: Yes
