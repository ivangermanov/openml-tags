This dataset contains twenty-two datasets from the book Visualizing Data published by Hobart Press. The dump was created using the data.dump function and can be read back into S by using the data.restore function. Each dataset has been given a unique name, which corresponds to the name used in the book.

To access the description of each dataset, refer to the index in the book under the entry "Data" followed by the name of the dataset. For instance, one of the datasets is called Barley. To learn more about this particular dataset, look up "Data, Barley" in the index.

The file containing this dataset can be found at ../data/visualizing/soil.csv.

Information about the Soil Dataset:
Classification Type: Numeric
Class Index: Not Applicable
Features: Northing, Easting, Resistivity, Track
