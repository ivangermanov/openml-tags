QSAR-TID-10904 Dataset

This dataset contains QSAR data showing activity values of several compounds on drug target TID: 10904. It has numerous rows and features, including IDs and class feature.

The features represent molecular fingerprints, which were generated from SMILES strings. These molecular fingerprints do not typically require missing value imputation.

The dataset is part of the MetaQSAR project, a comprehensive collection of QSAR datasets.

TID: 10904 corresponds to the protein target Serine/Threonine-Protein Kinase RAF, also known as cRaf or Proto-Oncogene c-RAF. This enzyme is involved in various cellular processes, including signal transduction, cell differentiation, and apoptosis.

Several small molecules have been identified as inhibitors of this enzyme, including Sorafenib, LY-3009120, XL-281, ARQ-736, and LGX-806. These compounds have been shown to inhibit the enzyme's activity, thereby affecting various cellular processes.

The dataset provides valuable information for researchers studying the structure-activity relationships of these compounds and their interactions with the Serine/Threonine-Protein Kinase RAF enzyme.
