The Compas Two Years dataset is categorized under Astronomy and Computer Systems and is part of the classification on both numerical and categorical features benchmark. This dataset has been transformed in the same manner as the one found in the tabular data benchmark.

Originally sourced from OpenML, this dataset contains features such as Sex, Age, Juvenile Misdemeanor Count, Prior Counts, and various categories for Age and Race. The target feature is Two Year Recidivism.

This dataset belongs to the classification on both numerical and categorical features benchmark.
