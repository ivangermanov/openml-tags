Name: Stylized Meta Album Textures Dataset

Tags: Textures dataset from the Describable Textures Dataset, stylized

Features: Category, File Name, File Path, Style, Style Name, Style Path, Stylized Path

This dataset is a collection of stylized textures, originating from the Describable Textures Dataset. It contains a range of textures, each with its unique characteristics, categorized and organized for easy access. The dataset provides a comprehensive set of features, including the category and file name of each texture, as well as its file path, style, and stylized path.
