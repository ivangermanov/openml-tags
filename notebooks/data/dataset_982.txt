Heart-C Dataset 

Tags: Chemistry, Life Science, Mythbusting, Study 

This dataset is a binarized version of the original dataset. The multi-class target feature has been converted into a two-class nominal target feature by relabeling the majority class as Positive and all others as Negative. 

The dataset features include patient information such as Age and Sex, as well as various medical measurements including Chest Pain, Resting Blood Pressure, Cholesterol Levels, and Electrocardiogram Results.

Keywords: Heart Disease, Medical Research, Data Conversion, Binary Classification
