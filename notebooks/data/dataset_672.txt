This dataset is part of a collection used in the book "A Casebook for a First Course in Statistics and Data Analysis," published by John Wiley and Sons, New York. The collection consists of multiple files, each representing a dataset analyzed in the book. The file names correspond to those used in the publication.

All data files are written in plain ASCII text format. Missing values are denoted by "M" across all datasets. Additional information about the datasets and the book can be accessed via gopher at swis.stern.nyu.edu or through the World Wide Web using a browser, starting from the URL address provided.

These datasets may be freely used for scientific, educational, and/or non-commercial purposes, provided appropriate acknowledgment is given by citing the aforementioned book reference.

The specific file discussed here is census.dat. The dataset is of numeric class type with no specific class index. Attribute names were automatically generated due to lack of information in the original data.

This collection serves as a valuable resource for students and researchers in statistics and data analysis, offering practical examples and real-world datasets to work with. It demonstrates the application of statistical methods to various fields and provides a hands-on approach to learning data analysis techniques.
