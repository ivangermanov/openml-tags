Colleges AAUP Dataset

Tags: Chemistry, Life Science, Mythbusting, Study 

This dataset is a binarized version of the original dataset. The numeric target feature has been converted to a two-class nominal target feature by calculating the mean and categorizing all instances with a lower target value as Positive and all others as Negative.

The dataset includes features such as Federal Identification Code, College Name, State, Institution Type, Average Salary for Full Professors, Average Compensation for Associate Professors, Number of Assistant Professors, and more.

Keywords: Colleges, Faculty Salaries, Academic Institutions, Higher Education, Data Analysis
