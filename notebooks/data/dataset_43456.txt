The IMDb Indonesian Movies Dataset contains information about Indonesian movies. The data was gathered using a web scraper and then converted and cleaned into a comma-separated values file.

Acknowledgments
This dataset is collected from IMDb dot com by utilizing a web scraper.

Content
There are numerous Indonesian movies in the dataset, consisting of eleven columns related to each movie. These columns comprise the title, release year, description, genre classification, rating, user ratings, number of votes, languages spoken, directors involved, actors featured, and runtime duration.
