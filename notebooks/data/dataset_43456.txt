IMDb Indonesian Movies Dataset

This dataset contains information about Indonesian movies gathered from IMDb.com. The data was collected using a web scraping tool and then converted and cleaned into a comma-separated values file.

Acknowledgements
This dataset was collected from IMDb.com.

Content
The dataset consists of over one thousand Indonesian movies, each with eleven columns of information. These columns include the title of the movie, the year it was released, a brief description, the genre, the rating, the users' rating, the number of votes, the languages spoken in the movie, the directors, the actors, and the runtime.

Features
The dataset includes information about the title, year, description, genre, rating, users' rating, votes, languages, directors, actors, and runtime of each movie.
