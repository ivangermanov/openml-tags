Last FM Dataset

Version One Point Zero (May Two Thousand Eleven)

Description

This dataset contains social networking, tagging, and music artist listening information from a set of users from Last.fm online music system. The dataset provides valuable insights into the behavior and preferences of music listeners, which can be used to develop more effective personalized recommendation systems.

License

The users' names and other personal information in Last.fm are not provided in the dataset to protect their privacy. The data contained in the dataset is made available for non-commercial use. Those interested in using the data in a commercial context should contact Last.fm staff.

Acknowledgments

This work was supported by the Spanish Ministry of Science and Innovation, and the Regional Government of Madrid.

References

When using this dataset, you should cite the Last.fm website. You may also cite the HetRec workshop as a reference.

Features

User, Artist, Sex, Country
