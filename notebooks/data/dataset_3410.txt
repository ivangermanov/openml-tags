Name: Lizards Dataset 
Tags: Graphical Models, Machine Learning, MaRDI, Mathematics, TA3 

Dataset Description 
Real-world data set about the perching behavior of two species of lizards in the South Bimini Island, which provides valuable insights into their habitat preferences.

Format of the Dataset 
The Lizards data set contains the following variables: 
Species - the species of the lizard, which is a two-level factor with levels referring to Sagrei and Distichus. 
Height - perch height, which is a two-level factor categorized as high, indicating greater than four point seven five feet, and low, indicating lesser or equal to four point seven five feet. 
Diameter - perch diameter, which is a two-level factor classified as narrow, indicating greater than four inches, and wide, indicating lesser or equal to four inches.

Source 
This dataset has been sourced from various publications focusing on graphical modeling and categorical data analysis, providing a comprehensive understanding of the complex relationships between these variables.
