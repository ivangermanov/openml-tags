The chscase Health dataset is a collection of data sets used in the book "A Casebook for a First Course in Statistics and Data Analysis". This dataset is part of a larger submission consisting of thirty-eight files, each representing a unique data set analyzed in the book. The names of the files correspond to the names used in the book.

The data files are written in plain ASCII character text. Missing values are represented by "Missing" in all data files. For more information about the data sets and the book, users can access the data through a gopher server at the address swis.stern.nyu.edu. The information is filed under Academic Departments and Research Centers, then Statistics and Operations Research, then Publications, then A Casebook for a First Course in Statistics and Data Analysis, and finally Welcome.

Alternatively, users can access the data from the World Wide Web using a web browser starting from the URL address http://www.stern.nyu.edu/Statistics and Operations Research/Casebook.

Users are free to use these datasets for scientific, educational, and non-commercial purposes, provided they give suitable acknowledgment.

File: Health Dataset

This dataset contains various attributes including Blood Pressure, Cholesterol Level, and other health-related factors.
