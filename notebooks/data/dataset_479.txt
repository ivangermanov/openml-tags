The Anal Cat Data collection is a comprehensive repository of data sets used in the book Analyzing Categorical Data. This submission includes a zip file containing two versions of each of eighty-four data sets, along with a readme file. Each data set is presented in both comma-delimited ASCII format and Microsoft Excel format.

Please note that these data sets are free to use for scientific, educational, and non-commercial purposes, provided that suitable acknowledgement is given by citing the reference.

Further information regarding the book, including details on statistical software, can be accessed at the website. 

Information about the dataset 
Classification Type: nominal 
Classification Index: last 

Features: Year, Pitcher, League, Type, Wins, Win Percentage, Saves, Earned Run Average, Strikeouts, Innings Pitched, Cy Young Award.

Keywords: categorical data, data analysis, statistics, data sets, research, education.
