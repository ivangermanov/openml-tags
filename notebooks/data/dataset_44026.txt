FIFA Tags: Dataset used in the Tabular Data Benchmark, transformed in the same way. This dataset belongs to the "Regression on Numerical Features" benchmark. Original source: FIFA Twenty-Two Complete Player Dataset. Please give credit to the original source if you use this dataset.

The dataset features include the player's age, height in centimeters, weight in kilograms, release clause in euros, club joined, and wage in euros.
