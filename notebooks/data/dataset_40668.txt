**Connect Four**

This database contains all legal eight-ply positions in the game of Connect Four in which neither player has won yet, and in which the next move is not forced. The attributes represent board positions on a six by six board. The outcome class is the game theoretical value for the first player, which can be a win, loss, or draw.

**Attribute Information**

The board is numbered like a standard grid, with letters a to g on the x-axis and numbers one to six on the y-axis. The values represent the state of each position on the board, which can be blank, taken by Player One, or taken by Player Two.

**Features**

The features of the dataset include the state of each position on the board, from the top left to the bottom right.

**Class**

The class attribute represents the game theoretical value for the first player, which can be a win, loss, or draw.
