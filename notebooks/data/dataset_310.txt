The Anal Cat Data Lawsuit dataset is a collection of data sets used in the book Analyzing Categorical Data. This dataset consists of a zip file containing two versions of each of eighty-four data sets, along with a README file. Each data set is presented in both comma-delimited ASCII format and Microsoft Excel format.

The Anal Cat Data Lawsuit dataset may be used freely for scientific, educational, and non-commercial purposes, provided suitable acknowledgement is given by citing the reference.

Further details regarding the book, including information on statistical software, are available at the website.

The dataset has a class type of nominal and the class index is the last feature. 

Features: Length of Service, Capital, Normalized, Minority Status, Layoff Status
