LastFM Dataset
Tags: Last FM Dataset

Version One Point Zero (May Two Thousand Eleven)

Description
This dataset contains social networking, tagging, and music artist listening information from a set of users from Last.fm online music system. The dataset is released in the framework of the International Workshop on Information Heterogeneity and Fusion in Recommender Systems at the ACM Conference on Recommender Systems. The dataset provides a unique opportunity to explore the relationships between users, artists, and tags in a real-world music streaming platform.

License
The users' names and other personal information in Last.fm are not provided in the dataset. The data contained in the dataset is made available for non-commercial use. Those interested in using the data in a commercial context should contact Last.fm staff.

Acknowledgements

References
When using this dataset, you should cite the Last.fm website. You may also cite the International Workshop on Information Heterogeneity and Fusion in Recommender Systems.

Features: user, artist, gender, country
