QSAR-TID-125 Dataset

This dataset contains QSAR data showing activity values of several compounds on drug target TID: 125. It has multiple rows and numerous features, including IDs and class feature.

The features represent molecular fingerprints, which were generated from SMILES strings using the Pipeline Pilot program.

Demographic information includes geographic location.

MetaQSAR project - September 2015

This dataset contains QSAR data showing activity values of several compounds on drug target TID: 125. 

Features include molecule ChEMBL ID and median PX50.
