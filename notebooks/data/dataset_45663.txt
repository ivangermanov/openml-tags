Name: Simulated Video Transcoding

Tags: See the GitHub repository for a description of this dataset.

Features: This dataset includes information about the duration of the video, the type of codec used, the width and height of the video, the bitrate, framerate, and the number of I, P, and B frames. Additionally, it includes the size of the I and P frames, the overall size of the video, and the output codec, bitrate, framerate, width, and height. The dataset also includes information about the user time.

Keywords: video transcoding, codec, bitrate, framerate, video size, output settings.
