The KDD IPUMS LA Ninety Seven Small dataset is categorized under Computer Systems and Machine Learning. It is part of the Tabular Data Benchmark and has undergone transformations similar to those in the original benchmark.

This dataset falls within the classification on numerical features benchmark. The original data was binarized, converting the multi-class target feature into a two-class nominal target feature. The majority class was labeled as Positive, denoted by 'P', while all other classes were labeled as Negative, denoted by 'N'.

The dataset features include Value, Rent, Total Income, Mother's Location, Family Size, Number of Children, Elderly Children, Young Children, Number of Siblings, Age, Occupation Score, Socioeconomic Index, Total Income, Wage Income, Business Income, Farm Income, Social Security Income, Welfare Income, Other Income, Poverty Level, and Binary Class.

Keywords: Tabular Data Benchmark, Classification, Numerical Features, Binarization, Machine Learning.
