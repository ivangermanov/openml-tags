Click Prediction Small 

Tags: Chemistry, Life Science 

This dataset provides information on predicting clicks on advertisements in a search engine. The data includes various features that describe the characteristics of the advertisement, such as the advertisement's title and description, as well as the user who viewed the advertisement. Additionally, the dataset includes information about the context in which the advertisement was displayed, including the query and keyword used to retrieve the results.

Features: click, impression, URL hash, advertisement ID, advertiser ID, depth, position, query ID, keyword ID, title ID, description ID, user ID

Keywords: click prediction, search engine, advertising, chemistry, life science
