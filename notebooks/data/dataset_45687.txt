Schizo Tags: 

The Schizophrenic Eye-Tracking Data dataset is a collection of information with a nominal class type, where the class index is the last feature. This dataset contains various features, including an identification number, target, and several gain ratios. Additionally, the dataset includes information about the sex of the individuals and a class designation.
