Click Dataset Tags: Features

This dataset contains various features related to online advertising, including the impression, which refers to the number of times an ad is displayed. The URL hash is also included, which is a unique identifier for the webpage where the ad is displayed. Additionally, the ad ID and advertiser ID are provided, which identify the specific ad and the company that created it. The depth and position of the ad on the webpage are also included, which can affect how visible the ad is to users. Furthermore, the query ID, keyword ID, title ID, and description ID are provided, which relate to the search query and ad content. Finally, the user ID and target are included, which identify the user who interacted with the ad and the target audience for the ad.

Keywords: online advertising, ad display, webpage identifier, ad visibility, search query, ad content, user interaction, target audience
