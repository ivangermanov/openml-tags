Turkish Super League Matches (1959-2021)

Tags: Context

This dataset is collected using a Python script and Selenium. The content of this dataset includes all of the matches played in the Turkish Super League, providing information about match days, half-time and full-time results, red cards seen by both teams, and more.

Racing Bar Chart

This dataset features a comprehensive range of information, including the date of each match, the season and week in which it was played, the home and visiting teams, the full-time score, the number of goals scored by each team, the division and tier of the league, the total number of goals scored, the goal difference, the result of the match, the half-time score, the number of goals scored by each team at half-time, the total number of goals scored at half-time, the half-time goal difference, the half-time result, the number of fans in attendance, whether the match was played at a neutral venue, and the number of red cards given to each team.

Keywords: Turkish Super League, football, matches, dataset, Python, Selenium, match days, results, red cards.
