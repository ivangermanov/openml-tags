Hungarian Chickenpox

A spatio-temporal dataset of weekly chickenpox cases from Hungary. This dataset features a date column, as well as columns for each of the following regions: Budapest, Baranya, Bacs, Bekes, Borsod, Csongrad, Fejer, Gyor, Hajdu, Heves, Jasz, Komarom, Nograd, Pest, Somogy, Szabolcs, Tolna, Vas, Veszprem, and Zala.

Scraped Data: Dataset Information

Additional Information

Interstate 94 Westbound Traffic Volume

This dataset contains hourly traffic volume data for the Interstate 94 Westbound traffic lane in Minnesota, specifically at the ATR station 301, which is roughly midway between Minneapolis and St Paul. The dataset also includes hourly weather features and holidays, which can impact traffic volume. There are no missing values in this dataset.

Additional Variable Information

The holiday variable is a categorical feature that includes US National holidays, as well as regional holidays, including the Minnesota State Fair. The temp variable is a numeric feature that represents the average temperature in Kelvin. The rain 1 hour variable is a numeric feature that represents the amount of rain that occurred in the hour in millimeters. The snow 1 hour variable is a numeric feature that represents the amount of snow that occurred in the hour in millimeters. The clouds all variable is a numeric feature that represents the percentage of cloud cover. The weather main variable is a categorical feature that provides a short textual description of the current weather. The weather description variable is a categorical feature that provides a longer textual description of the current weather. The date time variable is a datetime feature that represents the hour of the data collected in local CST time. The traffic volume variable is a numeric feature that represents the hourly I-94 ATR 301 reported westbound traffic volume.
