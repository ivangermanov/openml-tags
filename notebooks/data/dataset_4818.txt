Dataset used in the Tabular Data Benchmark, transformed in the same way. This dataset belongs to the Classification on Numerical Features benchmark.

Original Source: California Housing 

This dataset provides information about various factors affecting housing prices in California. The dataset includes several features that are relevant to understanding the complexities of the California real estate market.

The features included in this dataset are median income, house age, average rooms, average bedrooms, population, average occupation, latitude, longitude, and whether the price is above the median.
