The FIFA dataset is categorized under Computer Systems and Mathematics and is part of the regression on numerical features benchmark. It has been transformed in a specific manner for the purpose of the tabular data benchmark. The original source of this dataset can be found online.

This dataset consists of several key features, including player age, height in centimeters, weight in kilograms, and wage in euros. Additionally, it includes information about the player's club joining history and their release clause in euros.
