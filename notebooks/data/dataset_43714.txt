Context 
Some camera enthusiast went and described a vast collection of cameras based on thirteen distinct properties, providing a comprehensive overview of each device.

The dataset contains a detailed description of each camera, with the first row explaining the data type for each column, which can likely be removed.

The thirteen properties of each camera include the model, release date, maximum resolution, low resolution, effective pixels, zoom wide, zoom tele, normal focus range, macro focus range, storage included, weight including batteries, dimensions, and price.
