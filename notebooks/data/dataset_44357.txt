This dataset is a subsampling of the Credit dataset, with specific parameters set for the sampling process. The Credit dataset has been reduced to a more manageable size, with a maximum number of rows and columns, as well as a limited number of classes. 

The subsampling process involved uniformly selecting classes from the original dataset, followed by uniformly sampling columns. The resulting dataset has a specified number of rows, which were stratified accordingly to ensure representativity.

Features of this dataset include Revolving Utilization of Unsecured Lines, Age, Debt Ratio, Monthly Income, Number of Open Credit Lines and Loans, Number of Times Ninety Days Late, Number of Real Estate Loans or Lines, Number of Dependents, and Serious Delinquency in Two Years.
