The CNAE-9 Dataset: A Binarized Version for Agriculture and Sociology Studies 

This dataset is a binarized version of the original CNAE-9 dataset. It only considers instances with class labels one and two from the original dataset. 

Features included in this dataset are numerous variables that provide insight into agricultural practices and their impact on society. These variables range from various aspects of agricultural production to social and economic indicators related to agriculture.
