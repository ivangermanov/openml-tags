The Online Shoppers dataset consists of feature vectors belonging to individual sessions. This dataset was carefully constructed to ensure that each session belongs to a unique user within a one-year period, thereby avoiding any biases towards specific campaigns, special days, user profiles, or periods.

Source: https://archive.ics.uci.edu/ml/datasets/Online Shoppers Purchasing Intention Dataset

Features include bounce rates, exit rates, page values, whether the visit occurred on a special day, browser type, operating system, visitor type, administrative duration, month, region, product-related metrics, product-related duration, informational metrics, traffic type, informational duration, whether the visit occurred during the weekend, administrative metrics, and a classification label.

Keywords: online shoppers, purchasing intention, dataset, feature vectors, sessions, users, bias avoidance
