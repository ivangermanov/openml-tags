This data dump contains multiple datasets from the book Visualizing Data published by Hobart Press. The dump was generated using the data.dump function and can be restored into an S environment using the data.restore function. Each dataset in this dump corresponds to a specific dataset used in the book. 

To access a detailed description of each dataset, refer to the index section of the book under the heading 'Data' followed by the name of the dataset. For instance, if you want to learn more about the Barley dataset, simply look up 'Data, Barley' in the index.
