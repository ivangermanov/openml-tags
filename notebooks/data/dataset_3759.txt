Amazon Stock Data Latest

Tags: Computer Systems, Machine Learning

This dataset contains the values of the stock of Amazon, which date between October one and July fourteen. 

The content of this dataset includes seven different columns that comprise the date, the opening value of the stock, the closing value of the stock, volume, and several other essential pieces of information necessary to make accurate predictions.

Acknowledgments: 
This data was carefully extracted from Yahoo Finance, and one can also utilize their application programming interface to gain access to the data.

Features:
Date, Open, High, Low, Close, Adjusted Close, Volume
