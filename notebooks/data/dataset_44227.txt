The South Asian Churn Dataset 

Tags: Machine Learning, Meteorology 

This dataset utilizes real-life data collected from a major wireless telecommunications operator in South Asia. It comprises various features that provide valuable insights into customer behavior and preferences. These features include Network Age, Total Revenue, SMS Revenue, Data Revenue, Data Volume, Number of Calls, On-Network Revenue, Off-Network Revenue, Complaint Count, User Type for August, User Type for September, Favorite A for August, and Favorite A for September. The dataset also includes a Class feature, which categorizes customers based on their churn status.

Keywords: South Asia, Telecommunications, Customer Churn, Machine Learning, Dataset, Wireless Operator
