The South Asian Churn dataset is a real-life data collection used from a major wireless telecom operator in South Asia. This dataset is tagged as SATO. 

The features of this dataset include the age of the network, total revenue, SMS revenue, data revenue, data volume, total calls, on-net revenue, off-net revenue, complaint count, user type in August, user type in September, favorite A in August, favorite A in September, and the class of the data.
