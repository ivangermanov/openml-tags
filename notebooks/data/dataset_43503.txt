COVID-19 Cases by Country

Tags: Machine Learning, Mathematics

Context:
The COVID-19 pandemic is spreading rapidly across the globe, and this dataset helps analyze the extent to which it has affected different countries.

Content:
This dataset contains the total number of COVID-19 cases reported per country up until a certain point in time. It consists of six columns, including the date, location, total cases, new cases, total deaths, and new deaths.

Acknowledgements:
The data was collected from Our World in Data, a reliable source for coronavirus-related information.

Inspiration:
Let us try to create a bar chart race from this data to visualize the total cases reported by country over time.

Features:
Date, Location, Total Cases, New Cases, Total Deaths, New Deaths

Keywords: COVID-19, Pandemic, Country, Cases, Deaths, Data Analysis, Visualization
