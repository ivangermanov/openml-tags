Numerai Data Set 

Tags: Azure Pilot, Computational Universe, Machine Learning, Open ML CC 

Source: Kaggle 

The Numerai data set consists of cleaned, regularized, and encrypted global equity data. This data set has twenty-one columns of features, followed by a target column representing the binary class that needs to be predicted. The features are anonymous attributes that have been carefully selected to provide insights into stock market trends without revealing sensitive information.

Please cite the source of this data as Encrypted Stock Market Training Data from Numerai when using it for any purpose.

Keywords: machine learning, stock market, encrypted data, global equity, computational universe.
