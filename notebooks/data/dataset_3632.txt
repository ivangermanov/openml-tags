COVID-19 Rio de Janeiro (City)
Tags: Computer Systems, Machine Learning

Context - World Health Organization (WHO)
Coronavirus disease (COVID-19) is an infectious disease caused by a newly discovered coronavirus.

Content
This dataset contains information on the cumulative number of confirmed cases, deaths, and recoveries by neighborhood in the city of Rio de Janeiro, Brazil. It is essential to note that this is a time-series data, and therefore, the number of cases on any given day represents the total count up to that point. To obtain the number of new cases, one would need to calculate the difference between the current and previous days' counts. The data spans from April to December.

Dataset Inspiration 
The inspiration behind this dataset lies in analyzing the changes in the number of confirmed cases, deaths, and recoveries by neighborhood over time, as well as at the city level. Additionally, it aims to provide insights into the spread of the disease throughout the city.
