Ulaanbaatar Weather 2015-2020

Tags: Context

You can find a detailed weather data of Ulaanbaatar, the capital city of Mongolia, which spans from 2015 to 2020. This dataset includes timestamps in Coordinated Universal Time (UTC) and timely basis data of various weather-related features.

Inspiration

Since there is a significant need for weather data, but not enough free materials are available, this dataset is being shared to cater to that need.

Data Description

The dataset can be easily accessed on www.wunderground.com. It comprises the following features: expiration time, valid time, day indicator, temperature, weather icon, weather phrase, dew point, heat index, relative humidity, pressure, visibility, wind chill, wind direction, wind speed, ultraviolet description, feels like temperature, ultraviolet index, and cloud cover. Additionally, the dataset includes the date.
