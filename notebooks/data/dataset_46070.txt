Name: Stylized Meta Album Textures Dataset Extended

Tags: Textures dataset from the Describable Textures Dataset, stylized

Features: Category, File Name, File Path, Style, Style Name, Style Path, Stylized Path

This dataset is an extension of the original Describable Textures Dataset, featuring stylized textures. The dataset provides a comprehensive collection of textures, each with its unique characteristics and features. The category feature allows for grouping and classification of textures based on their properties. The file name and file path features provide information about the location and identity of each texture. The style feature captures the essence of each texture, while the style name and style path features offer additional context. Finally, the stylized path feature provides a transformed version of the original texture.

Keywords: Textures, Describable, Stylized, Dataset, Category, Style
