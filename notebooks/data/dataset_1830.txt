This dataset contains QSAR data showing activity values of several compounds on drug target Serine Threonine Protein Kinase SIK One, and it has multiple rows and features including IDs and class feature. The features represent Molecular Fingerprints which were generated from SMILES strings. They were obtained using the Pipeline Pilot program.

The dataset is part of the MetaQSAR project and was released in September. It should be cited when used.

The dataset provides information about the target Serine Threonine Protein Kinase SIK One, which is also known as Salt Inducible Kinase One. It is a single protein and is classified as an enzyme, kinase, and protein kinase belonging to the CAMK protein kinase group and CAMK protein kinase CAMK One family.

The dataset includes information about the expression of this target in different organisms, including Homo sapiens. It also provides information about the function of this target, including its involvement in various biological processes such as cell cycle, cell differentiation, and rhythmic process. Additionally, it includes information about its molecular function, including its ability to bind to various molecules and its enzymatic activity.
