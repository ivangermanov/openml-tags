**FOREX Eur/Dkk Hour Close**

Tags: Finance, Forex, Forex Close, Forex Hour, Transportation

This dataset contains the historical price data of the Forex Eur/Dk exchange rate from Dukascopy. Each instance represents a single candlestick for a one-hour period. The data ranges from January first to December thirteenth and excludes weekends, as the Forex market is not active on these days. The timestamp feature is set to the Europe/Amsterdam time zone.

The class attribute indicates the direction of the mean close price for the subsequent hour relative to the mean close price of the current hour. Specifically, the class attribute is true if the mean close price increases over the next hour and false if it decreases or remains unchanged.

The attributes included in this dataset are:

* Timestamp: The date and time of each data point in the Europe/Amsterdam time zone.
* Bid Open: The opening bid price for each time interval.
* Bid High: The highest bid price recorded during each time interval.
* Bid Low: The lowest bid price recorded during each time interval.
* Bid Close: The closing bid price for each time interval.
* Bid Volume: The number of times the bid price changed within each time interval.
* Ask Open: The opening ask price for each time interval.
* Ask High: The highest ask price recorded during each time interval.
* Ask Low: The lowest ask price recorded during each time interval.
* Ask Close: The closing ask price for each time interval.
* Ask Volume: The number of times the ask price changed within each time interval.
* Class: A binary indicator of whether the mean close price increases over the next hour.
