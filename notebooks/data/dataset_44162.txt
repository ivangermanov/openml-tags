Compass 

Tags: Machine Learning, Meteorology 

This dataset is part of the tabular data benchmark and has been transformed in the same manner. It falls under the classification on categorical and numerical features benchmark.

The original source of this dataset is publicly available.

Features include demographic information such as Sex and Race, as well as criminal history details including Juvenile Felony Count, Prior Count, and Days between Screening Arrest. Additionally, there are assessment scores including Decile Score and Type of Assessment.

Please provide credit to the original source when utilizing this dataset.
