Medical Charges 

Tags: Demographics, Economics 

Dataset used in the Tabular Data Benchmark, transformed in the same way. This dataset belongs to the "Regression on Numerical Features" benchmark.

The Inpatient Utilization and Payment Public Use File provides information on inpatient discharges for Medicare fee-for-service beneficiaries. It offers insights into utilization, payment (total payment and Medicare payment), and hospital-specific charges for the thousands of US hospitals that receive Medicare Inpatient Prospective Payment System payments.

The file is organized by hospital and Medicare Severity Diagnosis Related Group, covering fiscal years. It provides a comprehensive overview of the healthcare system, enabling analysis of trends and patterns in hospital utilization and payment.

Features: total discharges, average covered charges, average medicare payments, average total payments

Keywords: Medicare, hospital charges, inpatient discharges, payment systems, healthcare analysis
