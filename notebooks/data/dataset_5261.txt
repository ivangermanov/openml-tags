Dataset Name: Padding Attack Dataset - OpenSSL111t
Tags: Bleichenbacher Padding Attack

This dataset was created to study the Bleichenbacher Padding Attack on servers using OpenSSL111t.

Attribute Names:
The dataset contains attributes related to TCP packets and TLS records.

Features:

* Label
* Source Port
* Destination Port
* Port
* Stream
* Length
* Sequence Number
* Next Sequence Number
* Acknowledgment Number
* Header Length
* Flags - Reserved
* Flags - Nonce Sum
* Flags - Congestion Window Reduced
* Flags - Explicit Congestion Notification
* Flags - Urgent
* Flags - Acknowledgment
* Flags - Push Function
* Flags - Reset Connection
* Flags - Synchronize Sequence Numbers
* Flags - Finish Sending Data
* Window Size Value
* Window Size
* Window Size Scale Factor
* Checksum Status
* Urgent Pointer
* Options - No Operation
* Option Kind
* Option Length
* Timestamp Value
* Time Delta
* Record Content Type
* Record Length
* Alert Message Level
* Alert Message Description
* Order

Similarly, attributes for two discarded packets, DISC0 and DISC1, are also included in the dataset.

Keywords: Bleichenbacher Padding Attack, OpenSSL111t, TCP Packets, TLS Records, Cybersecurity.
