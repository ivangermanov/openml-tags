Wine Reviews 

Tags: Machine Learning, Medicine 

This wine data set was gathered from Kaggle and consists of information scraped from Wine Enthusiast magazine during a specific period of time. The code used for web scraping can be accessed through GitHub.

The features of this data set include the country of origin, wine description, wine designation, rating points, price, province, regional information, grape variety, and winery.

Keywords: Wine Enthusiast, Kaggle, Machine Learning, Web Scraping, Wine Data
