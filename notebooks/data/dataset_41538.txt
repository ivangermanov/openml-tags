Conference Attendance

Tags: Data set shows information about participants of a mathematics conference.

The dataset contains information about the participants of a mathematics conference, with the target column being "Is Present" for a classification task. The features included in the dataset are University, T-Shirt Size, Favorite Subject, Is Present, and Participation, as well as Registration Date Category.

Keywords: mathematics conference, participant information, classification task, university, t-shirt size, favorite subject.
