**Dataset Description**

The data contains information on superconductors. The first eighty-one columns contain extracted features, and the eighty-second column contains the critical temperature, which is used as the target variable. The original data from which the features were extracted comes from a public source, specifically the website http://supercon.nims.go.jp/index_en.html.

**Related Studies**

This dataset is related to studies on superconductivity, including a study published in Frontiers in Materials.

**Source**

The dataset originates from the University of California, Irvine (UCI) Machine Learning Repository.

**Citation**

Please cite the UCI Machine Learning Repository when using this dataset.

The dataset features include various properties of superconductors, such as atomic mass, atomic radius, density, electron affinity, fusion heat, thermal conductivity, and valence.
