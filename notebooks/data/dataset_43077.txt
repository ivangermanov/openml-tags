Art Daily Jumps Up Dataset

Tags: Artificial with Anomaly

Features: Time Stamp, Value, Anomaly Detection

This dataset, Art Daily Jumps Up, is characterized by its artificial nature and the presence of anomalies. It contains a time stamp feature, which records the timing of each data point, as well as a value feature, which represents the actual data being measured. Additionally, the dataset includes an anomaly detection feature, which identifies unusual patterns or outliers in the data.

Keywords: Artificial Data, Anomaly Detection, Time Series, Data Analysis
