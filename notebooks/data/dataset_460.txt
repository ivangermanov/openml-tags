A collection of data sets used in the book "Analyzing Categorical Data," published by Springer-Verlag, New York, in 2003. The submission consists of a zip file containing two versions of each of 84 data sets, plus a README file. Each data set is provided in comma-delimited ASCII (.csv) format and Microsoft Excel (.xls) format.

These data sets may be used freely for scientific, educational, and/or noncommercial purposes, provided suitable acknowledgment is given by citing the aforementioned reference. Further details concerning the book, including information on statistical software (such as sample S-PLUS/R and SAS code), are available at the specified website.

Information about the dataset:
Class Type: nominal
Class Index: none specific

Note: Quotes, Single-Quotes, and Backslashes were removed, and Blanks were replaced with Underscores.

Features: Film, Roger Ebert, Jeffrey Lyons, Michael Medved, Rex Reed, Gene Shalit, Joel Siegel, Gene Siskel, Peter Travers
