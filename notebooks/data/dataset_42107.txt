The HMEQ-P Dataset is categorized under Computer Systems and Machine Learning. This dataset comprises various features including Loan, Mortgage Due, Property Value, Reason for Loan, Job Type, Years on Job, Derogatory Marks, Delinquency, Credit Age, Number of Inquiries, Credit Limit Number, Debt-to-Income Ratio, and Data Index.

Keywords: HMEQ-P, Loan, Mortgage, Credit, Machine Learning
