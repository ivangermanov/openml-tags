Fri C One Dataset: This dataset is a binarized version of the original dataset, where the numeric target feature has been converted into a two-class nominal target feature. This was achieved by computing the mean and then classifying all instances with a lower target value as Positive and all others as Negative.

This dataset belongs to the categories of Chemistry, Life Science, Mythbusting, Study, and features fifty variables, including ozone one through ozone fifty.

The Binary Class feature has been included to provide a clear classification of the instances.

Keywords: Dataset, Binarized, Nominal Target Feature, Chemistry, Life Science, Mythbusting, Study.
