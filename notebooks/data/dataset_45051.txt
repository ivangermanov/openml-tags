The Adult Test Dataset 

Tags: Computer Systems, Machine Learning 

This dataset includes various features related to an individual's demographic and employment information. The features include Age, which represents the person's age; Education Number, which indicates the number of years of education; Capital Gain, which refers to the profit made from selling an investment; Capital Loss, which refers to the loss incurred from selling an investment; Hours per Week, which denotes the number of hours worked per week.

Additionally, this dataset includes categorical features such as Work Class, Education, Marital Status, Occupation, Relationship, Race, Sex, and Native Country, which provide insights into an individual's background and lifestyle. The Class feature represents the target variable.

Keywords: Demographic Information, Employment Data, Machine Learning, Computer Systems, Education Level, Investment Profit/Loss.
