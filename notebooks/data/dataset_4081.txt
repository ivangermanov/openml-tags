The FIFA dataset is categorized under Computer Systems and Physical Sciences, and is part of the regression on numerical features benchmark. This dataset has been transformed in a specific way and is available on GitHub as part of the Tabular Benchmark repository.

The original source of this dataset is a complete player dataset, which provides comprehensive information about football players. If you utilize this dataset, please ensure you give proper credit to the original source.

This dataset comprises various features, including the age of the players, their height in centimeters, weight in kilograms, and the amount of their release clause in euros. Additionally, it includes details about the clubs they have joined and their wages in euros.
