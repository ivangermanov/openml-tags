US Open Men 2013

Tags: Images, Life Science

This dataset is part of a collection of eight files containing match statistics for both women and men at the four major tennis tournaments of the year 2013. Each file has forty-two columns and a minimum of seventy-six rows. This file contains the statistics for the US Open Men's Edition for 2013.

Attribute Information:

Player One Name refers to the name of the first player.
Player Two Name refers to the name of the second player.
Result indicates the outcome of the match, where one represents a win for Player One and zero represents a loss.
First Serve Percentage for Player One is the percentage of successful first serves made by Player One.
First Serve Won by Player One is the number of first serves won by Player One.
Second Serve Percentage for Player One is the percentage of successful second serves made by Player One.
Second Serve Won by Player One is the number of second serves won by Player One.
Aces Won by Player One represents the number of aces served by Player One.
Double Faults Committed by Player One represents the number of double faults made by Player One.
Winners Earned by Player One represents the number of winners hit by Player One.
Unforced Errors Committed by Player One represents the number of unforced errors made by Player One.
Break Points Created by Player One represents the number of break points created by Player One.
Break Points Won by Player One represents the number of break points won by Player One.
Net Points Attempted by Player One represents the number of net points attempted by Player One.
Net Points Won by Player One represents the number of net points won by Player One.
Total Points Won by Player One represents the total number of points won by Player One.
Set One Result for Player One represents the result of the first set for Player One.
Set Two Result for Player One represents the result of the second set for Player One.
Set Three Result for Player One represents the result of the third set for Player One.
Set Four Result for Player One represents the result of the fourth set for Player One.
Set Five Result for Player One represents the result of the fifth set for Player One.
Final Number of Games Won by Player One represents the final number of games won by Player One.

The same attributes are repeated for Player Two.

Round represents the round of the tournament at which the game was played.

Features:
Player One, Player Two, Round, Result, Final Number of Games Won by Player One, Final Number of Games Won by Player Two, First Serve Percentage, First Serve Won, Second Serve Percentage, Second Serve Won, Aces Won, Double Faults Committed, Winners Earned, Unforced Errors Committed, Break Points Created, Break Points Won, Net Points Attempted, Net Points Won, Total Points Won, Set One Result, Set Two Result, Set Three Result, Set Four Result, Set Five Result

Dataset Information:
Has Missing Values? Yes

Keywords: Tennis, US Open, Men's Edition, Match Statistics, Tournament Results
