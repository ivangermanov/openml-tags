**usopen-men-2013**
**Tags:** Tennis Major Tournament Match Statistics

This dataset is part of a collection of eight files containing match statistics for both women and men at the four major tennis tournaments of the year 2013. Each file has forty-two columns and a minimum of seventy-six rows. This file contains the statistics for the US Open Men's Edition for 2013.

### Attribute Information

* Player One: Name of Player One
* Player Two: Name of Player Two
* Result: Result of the match (one if Player One wins, zero otherwise)
* First Serve Percentage for Player One: Real number representing the percentage of first serves won by Player One
* First Serve Won by Player One: Real number representing the number of first serves won by Player One
* Second Serve Percentage for Player One: Real number representing the percentage of second serves won by Player One
* Second Serve Won by Player One: Real number representing the number of second serves won by Player One
* Aces Won by Player One: Integer representing the number of aces won by Player One
* Double Faults Committed by Player One: Integer representing the number of double faults committed by Player One
* Winners Earned by Player One: Real number representing the number of winners earned by Player One
* Unforced Errors Committed by Player One: Real number representing the number of unforced errors committed by Player One
* Break Points Created by Player One: Real number representing the number of break points created by Player One
* Break Points Won by Player One: Real number representing the number of break points won by Player One
* Net Points Attempted by Player One: Real number representing the number of net points attempted by Player One
* Net Points Won by Player One: Real number representing the number of net points won by Player One
* Total Points Won by Player One: Real number representing the total points won by Player One
* Set One Result for Player One: Integer representing the result of set one for Player One
* Set Two Result for Player One: Integer representing the result of set two for Player One
* Set Three Result for Player One: Integer representing the result of set three for Player One
* Set Four Result for Player One: Integer representing the result of set four for Player One
* Set Five Result for Player One: Integer representing the result of set five for Player One
* Final Number of Games Won by Player One: Integer representing the final number of games won by Player One
* First Serve Percentage for Player Two: Real number representing the percentage of first serves won by Player Two
* First Serve Won by Player Two: Real number representing the number of first serves won by Player Two
* Second Serve Percentage for Player Two: Real number representing the percentage of second serves won by Player Two
* Second Serve Won by Player Two: Real number representing the number of second serves won by Player Two
* Aces Won by Player Two: Integer representing the number of aces won by Player Two
* Double Faults Committed by Player Two: Integer representing the number of double faults committed by Player Two
* Winners Earned by Player Two: Real number representing the number of winners earned by Player Two
* Unforced Errors Committed by Player Two: Real number representing the number of unforced errors committed by Player Two
* Break Points Created by Player Two: Real number representing the number of break points created by Player Two
* Break Points Won by Player Two: Real number representing the number of break points won by Player Two
* Net Points Attempted by Player Two: Real number representing the number of net points attempted by Player Two
* Net Points Won by Player Two: Real number representing the number of net points won by Player Two
* Total Points Won by Player Two: Real number representing the total points won by Player Two
* Set One Result for Player Two: Integer representing the result of set one for Player Two
* Set Two Result for Player Two: Integer representing the result of set two for Player Two
* Set Three Result for Player Two: Integer representing the result of set three for Player Two
* Set Four Result for Player Two: Integer representing the result of set four for Player Two
* Set Five Result for Player Two: Integer representing the result of set five for Player Two
* Final Number of Games Won by Player Two: Integer representing the final number of games won by Player Two
* Round: Integer representing the round of the tournament at which the game is played
