Flora Tags: 

The Flora dataset originates from the first ChaLearn AutoML challenge, which took place in two thousand fourteen. This dataset only includes the training data, as there were no labels provided for the validation and test sets. 

The dataset features a target variable, accompanied by a multitude of other variables, which are not explicitly named. 

Dataset from the first ChaLearn Auto Machine Learning challenge.
