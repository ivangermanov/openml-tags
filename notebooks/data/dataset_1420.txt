Name: CPU with Vendor

Tags: Features, Vendor, Minimum Memory, Maximum Memory, Cache, Minimum Channels, Maximum Channels, Class

This dataset contains information about central processing units (CPUs) from various vendors. It includes details on CPU features, memory specifications, cache sizes, and channel configurations. The data encompasses a range of CPU models with their respective minimum and maximum memory capacities, as well as cache sizes. Additionally, the dataset provides information on the minimum and maximum number of channels supported by each CPU. The classification of CPUs into different classes is also included, offering insights into their performance categories.

Keywords: CPU, Vendor, Memory, Cache, Channels, Classification
