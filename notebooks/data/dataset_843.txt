House Eight L Dataset 

Tags: Binarized Regression Problem, Chemistry, Life Science, Mythbusting One, Study One, Study Fifteen, Study Twenty, Study Forty-One, Study Seven 

This dataset is a binarized version of the original data set. The numeric target feature has been converted into a two-class nominal target feature. This was achieved by computing the mean and then classifying all instances with a lower target value as Positive and all others as Negative. 

The features included in this dataset are as follows: P Three, P Six Point Four, P Eleven Point Three, P Sixteen Point Two, P Nineteen Point Two, H Five Point Two, H Fifteen Point One, H Forty Point Four, Binary Class.
