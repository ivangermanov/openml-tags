Toxic Comment Dataset is categorized under Machine Learning and Physical Sciences. This dataset contains comments along with their respective labels. Each comment has been labeled for toxicity, severity of toxicity, obscenity, threats, insults, and identity-based hate speech.

The dataset features include an identification number for each comment, the actual comment text, and labels indicating whether the comment is toxic, severely toxic, obscene, threatening, insulting, or promotes hatred towards an individual or group based on their identity.

Keywords: Toxic Comments, Machine Learning, Hate Speech, Insults, Threats
