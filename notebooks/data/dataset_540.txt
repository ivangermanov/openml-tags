This file contains the data in "The MU284 Population" from Appendix B of the book "Model Assisted Survey Sampling" published by Springer-Verlag, New York. The data set contains observations on variables, plus a line with variable names. Please consult the mentioned appendix for more information about this data set. The data were scanned from the book and interpreted with OCR technique. Please note that errors may occur in such a process. The result was macro edited against "The Clustered MU284 Population" in Appendix C of the book. Please use the data at your own risk - no responsibility is taken for any problems eventual remaining errors will cause you.

Several typos in the first printing of the book have been corrected, including adjustments to values for ME84 and RMT85 for specific labels. The data was submitted to StatLib with the permission of Springer-Verlag.

Information about the dataset:
CLASSTYPE: numeric
CLASSINDEX: none specific
Features: LABEL, P85, P75, RMT85, CS82, SS82, S82, ME84, REV84, REG, CL

The dataset contains population data that can be used for survey sampling research and analysis. It provides a comprehensive set of variables related to demographics, economics, and regional classifications. Researchers and statisticians can utilize this data to develop and test survey sampling methodologies, as well as to study population characteristics and trends.
