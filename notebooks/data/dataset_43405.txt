**Turkey Covid 19 Dataset**

Tags: Computer Systems, Machine Learning

Context: The Turkey Covid 19 dataset contains information about the COVID-19 pandemic in Turkey.

Data Source: The data is sourced from the official website of the Ministry of Health of the Republic of Turkey.

Content:
This dataset has been created in accordance with the data shared by the Ministry of Health of the Republic of Turkey. Daily updates are made to the dataset using data mining methods, where new data is extracted from the Ministry of Health's website. The updated dataset is then shared on GitHub.

The dataset includes the following columns:

* Total Tests: The cumulative number of tests conducted up to each day.
* Total Cases: The cumulative number of confirmed cases announced up to each day.
* Total Deaths: The cumulative number of reported deaths up to each day.
* Total Intensive Care: The cumulative number of individuals in intensive care units announced up to each day.
* Total Intubated: The cumulative number of individuals who have been intubated announced up to each day.
* Total Recovered: The cumulative number of individuals who have recovered from the virus.
* Daily Tests: The number of tests conducted on a daily basis.
* Daily Cases: The number of new confirmed cases reported on a daily basis.
* Daily Deaths: The number of new reported deaths on a daily basis.
* Daily Recovered: The number of new recoveries reported on a daily basis.
