FOREX EUR CHF Day Close Tags: Demographics, Finance, Forex, Forex Close, Forex Day, Health 

This dataset contains the historical price data of the Forex Euro versus Swiss Franc exchange rate, sourced from Dukascopy Historical Data Feed. Each instance represents one daily candlestick. The entire dataset spans a specific period, excluding weekends when Forex markets are closed.

The timestamp feature is set to the Europe Amsterdam timezone. The class attribute indicates the direction of the mean close price between the current and following days. Specifically, the class attribute is true if the mean close price increases on the subsequent day, and false if it decreases or remains unchanged.

Attributes:

* Timestamp: The time of the current data point, set to the Europe Amsterdam timezone.
* Bid Open: The bid price at the start of the daily time interval.
* Bid High: The highest bid price recorded during this daily time interval.
* Bid Low: The lowest bid price recorded during this daily time interval.
* Bid Close: The bid price at the end of the daily time interval.
* Bid Volume: The number of times the bid price changed within this daily time interval.
* Ask Open: The ask price at the start of the daily time interval.
* Ask High: The highest ask price recorded during this daily time interval.
* Ask Low: The lowest ask price recorded during this daily time interval.
* Ask Close: The ask price at the end of the daily time interval.
* Ask Volume: The number of times the ask price changed within this daily time interval.
* Class: Whether the average price will increase on the subsequent day.

Keywords: Forex, Euro, Swiss Franc, Exchange Rate, Candlestick, Timezone, Class Attribute, Mean Close Price.
