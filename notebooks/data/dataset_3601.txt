QSAR-TID-13000 Dataset 

This dataset contains QSAR data showing activity values of several compounds on drug target Matrix Metalloproteinase One. It has numerous rows and features, including identifiers and class features. The features represent molecular fingerprints, which were generated from SMILES strings using the Pipeline Pilot program.

The dataset is part of the MetaQSAR project, which aims to provide a comprehensive repository of QSAR data for various biological targets. Matrix Metalloproteinase One is an enzyme involved in the breakdown of extracellular matrices, and inhibitors of this enzyme have potential therapeutic applications.

The dataset includes information about the compounds, such as their molecular structure, and their activity against Matrix Metalloproteinase One. The activity values are expressed in terms of pseudo-pCI50, which is a measure of the potency of the compound.

The dataset also includes information about the target protein, Matrix Metalloproteinase One, such as its synonyms, classification, and description. Additionally, it provides information about related proteins, such as other Matrix Metalloproteinases, and their inhibitors.

Keywords: QSAR, Matrix Metalloproteinase One, enzyme inhibitors, biological targets, MetaQSAR project.
