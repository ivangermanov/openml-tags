Attribute Descriptions:

The dataset contains attributes related to the Change Cipher Spec TCP Acknowledgement, TLS Alert, and TCP Disconnect messages.

The Change Cipher Spec TCP Acknowledgement attributes include:

* TCP Source Port
* TCP Destination Port
* TCP Port
* TCP Stream Index
* TCP Segment Length
* TCP Sequence Number
* TCP Next Sequence Number
* TCP Acknowledgment Number
* TCP Header Length
* TCP Flags including Reserved, Nonce, Congestion Window Reduced, ECN-Echo, Urgent, Acknowledgment, Push, Reset, Syn, and Fin
* TCP Window Size Value
* TCP Calculated Window Size
* TCP Window Size Scaling Factor
* TCP Checksum Status
* TCP Urgent Pointer
* TCP Options including NOP, Option Kind, Option Length, and Timestamp Value
* Time Delta since Previous Frame in this TCP Stream
* Message Order within Server Responses

* TLS Record Content Type
* TLS Record Length

* TLS Alert Message Description
