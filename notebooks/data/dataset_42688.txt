Brazilian Houses 

Tags: Chemistry, Machine Learning 

This dataset contains houses to rent with thirteen different features. Please note that some values in the dataset can be considered outliers for further analysis. It is essential to bear in mind that the web crawler was used solely to gather the data; therefore, it is possible that errors in the original data exist.

Changes in Data Between Versions of Dataset 
Since the web crawler was run on different days for each version of the dataset, there may be differences such as added or deleted houses (as well as added cities).

Notes 
This dataset corresponds to the second version of the original dataset. The value 'hyphen' in the attribute floor was replaced by 'zero' as it refers to houses with just one floor.

Features 
The dataset includes information about the city, area, number of rooms, number of bathrooms, parking spaces, floor, whether animals are allowed, furniture, Homeowners Association fee in Brazilian Real, rent amount in Brazilian Real, property tax in Brazilian Real, fire insurance in Brazilian Real, and total cost in Brazilian Real.
