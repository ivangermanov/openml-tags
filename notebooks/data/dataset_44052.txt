The Compass dataset, categorized under Computer Systems and Mathematics, is part of the tabular data benchmark. It belongs to the classification on categorical and numerical features benchmark. 

This dataset has been transformed in the same manner as the original dataset found in the GitHub repository for the tabular benchmark. The original source of the dataset is credited to the Compass Analysis project.

The Compass dataset comprises several features including Sex, Age, Age Category, Race, Juvenile Felony Count, Juvenile Misdemeanor Count, Juvenile Other Count, Priors Count, Days Between Screening and Arrest, Days from Compass, Charge Degree, Decile Score, Score Text, Type of Assessment, Decile Score, Score Text, End, and Is Recid.

Keywords: Compass dataset, Computer Systems, Mathematics, Classification, Categorical Features, Numerical Features, Tabular Data Benchmark.
