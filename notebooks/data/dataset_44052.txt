Compass Dataset

Tags: Dataset used in the Tabular Data Benchmark, transformed in the same way. This dataset belongs to the "Classification on Categorical and Numerical Features" benchmark.

Original Source: The dataset originates from the ProPublica Compass Analysis.

Features: The dataset includes the following features: Sex, Age, Age Category, Race, Juvenile Felony Count, Juvenile Misdemeanor Count, Juvenile Other Count, Prior Counts, Days Between Screening and Arrest, Days from Compass, Charge Degree, Decile Score, Score Text, Type of Assessment, Decile Score, Score Text, End, and Is Recidivism.

Keywords: Compass, Tabular Data, Classification, Categorical Features, Numerical Features, Benchmark, ProPublica.
