Timing Attack Dataset Thirty Micro Seconds Delay 

Tags: Chemistry, Life Science Bleichenbacher 

This dataset is related to the Bleichenbacher Timing Attack with a delay of thirty micro seconds, created on a specific date.

Attribute Descriptions:

The CCS zero attributes describe the TCP source port, destination port, port, stream, length, sequence number, next sequence number, acknowledgment number, header length, reserved flags, nonce flags, congestion window reduced flags, ECN echo flags, urgent flags, acknowledgment flags, push flags, reset flags, syn flags, fin flags, window size value, calculated window size, window size scale factor, checksum status, urgent pointer, options nop, option kind, option length, timestamp tsval, time delta, and message order of the first change cipher spec TCP acknowledgement.

The TLS zero attributes describe the TCP source port, destination port, port, stream, length, sequence number, next sequence number, acknowledgment number, header length, reserved flags, nonce flags, congestion window reduced flags, ECN echo flags, urgent flags, acknowledgment flags, push flags, reset flags, syn flags, fin flags, window size value, calculated window size, window size scale factor, checksum status, urgent pointer, options nop, option kind, option length, timestamp tsval, time delta, TLS record content type, TLS record length, TLS alert message level, TLS alert message description, and message order of the first TLS alert.

The DISC zero attributes describe the TCP source port, destination port, port, stream, length, sequence number, next sequence number, acknowledgment number, header length, reserved flags, nonce flags, congestion window reduced flags, ECN echo flags, urgent flags, acknowledgment flags, push flags, reset flags, syn flags, fin flags, window size value, calculated window size, window size scale factor, checksum status, urgent pointer, options nop, option kind, option length, timestamp tsval, time delta, and message order of the first TCP disconnect.

The DISC one attributes describe the TCP source port, destination port, port, stream, length, sequence number, next sequence number, acknowledgment number, header length, reserved flags, nonce flags, congestion window reduced flags, ECN echo flags, urgent flags, acknowledgment flags, push flags, reset flags, syn flags, fin flags, window size value, calculated window size, window size scale factor, checksum status, urgent pointer, options nop, option kind, option length, timestamp tsval, time delta, and message order of the second TCP disconnect.

Features include label, CCS zero attributes, TLS zero attributes, DISC zero attributes, and DISC one attributes.
