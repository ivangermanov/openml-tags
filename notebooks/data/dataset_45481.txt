Timing Attack Dataset Ten Micro Seconds Delay 

Tags: Bleichenbacher Timing Attack 

This dataset contains information about a ten micro seconds delay in a Bleichenbacher timing attack. The dataset was created on September Fifteenth.

Attribute Descriptions:

The attributes in this dataset describe various aspects of the Change Cipher Spec TCP Acknowledgement, TLS Alert, and TCP Disconnect messages.

CCS Zero describes the first Change Cipher Spec TCP Acknowledgement message. It includes attributes such as source port, destination port, source or destination port, TCP stream index, TCP segment length, sequence number, next sequence number, acknowledgment number, header length, reserved flag, nonce flag, congestion window reduced flag, echo congestion notification flag, urgent flag, acknowledgment flag, push flag, reset flag, synchronization flag, finish flag, window size value, calculated window size, window size scale factor, checksum status, urgent pointer, no operation option, option kind, option length, timestamp value, time delta, and message order.

TLS Zero describes the first TLS Alert message. It includes attributes such as source port, destination port, source or destination port, TCP stream index, TCP segment length, sequence number, next sequence number, acknowledgment number, header length, reserved flag, nonce flag, congestion window reduced flag, echo congestion notification flag, urgent flag, acknowledgment flag, push flag, reset flag, synchronization flag, finish flag, window size value, calculated window size, window size scale factor, checksum status, urgent pointer, no operation option, option kind, option length, timestamp value, time delta, TLS record content type, TLS record length, TLS alert message level, TLS alert message description, and message order.

DISC Zero describes the first TCP Disconnect message. It includes attributes such as source port, destination port, source or destination port, TCP stream index, TCP segment length, sequence number, next sequence number, acknowledgment number, header length, reserved flag, nonce flag, congestion window reduced flag, echo congestion notification flag, urgent flag, acknowledgment flag, push flag, reset flag, synchronization flag, finish flag, window size value, calculated window size, window size scale factor, checksum status, urgent pointer, no operation option, option kind, option length, timestamp value, time delta, and message order.

DISC One describes the second TCP Disconnect message. It includes attributes such as source port, destination port, source or destination port, TCP stream index, TCP segment length, sequence number, next sequence number, acknowledgment number, header length, reserved flag, nonce flag, congestion window reduced flag, echo congestion notification flag, urgent flag, acknowledgment flag, push flag, reset flag, synchronization flag, finish flag, window size value, calculated window size, window size scale factor, checksum status, urgent pointer, no operation option, option kind, option length, timestamp value, time delta, and message order.
