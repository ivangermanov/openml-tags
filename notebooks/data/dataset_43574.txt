The Marginal Revolution Blog Post Data set contains information on blog posts from MarginalRevolution.com, spanning from January first, two thousand and ten to September seventeenth, two thousand and sixteen. For each post, the following attributes are collected: the post title, the date the post was published, the content of the post in words, the total number of words in the post, and the number of comments on the post. Additionally, the data set includes dummy variables for several commonly used categories.

The data was gathered using Python's Beautiful Soup package and cleaned using the R programming language. The Python and R code used for this process can be found on the GitHub page.

The data set features a range of categories, including Current Affairs, Education, Music, Philosophy, Political Science, Science, History, Law, Games, Books, Food and Drink, Data Source, Web Technology, Economics, and Medicine.
