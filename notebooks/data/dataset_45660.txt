Name: Simulated Electricity Dataset

Tags: See the GitHub repository for a description of this dataset, which is related to Confidence Intervals for Generalized Estimating Equations.

Features: This dataset includes information on the date, day of the week, and period of the day, as well as the New South Wales electricity price and demand, Victoria electricity price and demand, and electricity transfer between the two regions. The dataset is categorized into a class.

Keywords: Electricity, Prices, Demand, Transfer, Regions, Classification
