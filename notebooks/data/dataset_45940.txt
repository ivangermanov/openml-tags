Login Data for Cybersecurity Events

Tags: Login Data for Cybersecurity Events

Features: Timestamp, Message, Source IP Address, Event Identification, Target User Name, Task, Identification, Label

This dataset contains login data for cybersecurity events, which is essential for monitoring and analyzing system access. The features included in this dataset provide a comprehensive overview of each login event, including the timestamp, message, source IP address, event identification, target user name, task, identification, and label.
