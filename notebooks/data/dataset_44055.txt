The Analcatdata Supreme dataset belongs to the regression on categorical and numerical features benchmark. It is a collection of datasets originally used in the book Analyzing Categorical Data. The dataset is provided in a zip file containing two versions of each dataset, one in comma-delimited ASCII format and the other in Microsoft Excel format.

The datasets may be used freely for scientific, educational, and non-commercial purposes, provided suitable acknowledgement is given by citing the original reference. Further details concerning the book, including information on statistical software, are available at the website.

The Analcatdata Supreme dataset does not have a specific class type or class index. The dataset features include actions taken, liberal, precedent alteration, unanimous decisions, year of decision, lower court disagreement, and log exposure.
