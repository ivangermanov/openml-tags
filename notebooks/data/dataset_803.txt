Heart-H Dataset 

Tags: Chemistry, Life Science 

This dataset is a binarized version of the original data set. The multi-class target feature has been converted to a two-class nominal target feature by relabeling the majority class as Positive and all others as Negative. 

The original dataset has been transformed to create this new version. 

Features include: Age, Sex, Chest Pain, Trest Blood Pressure, Cholesterol, Fasting Blood Sugar, Resting Electrocardiogram, Maximum Heart Rate Achieved, Exercise Induced Angina, Old Peak, Slope of the Peak Exercise, Number of Major Vessels Colored, Thallium Stress Test.

Keywords: Heart-H, Chemistry, Life Science, dataset, binarized, target feature, Positive, Negative
