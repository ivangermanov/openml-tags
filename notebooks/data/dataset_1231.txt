The Heart Dataset 

Tags: Chemistry, Life Sciences 

This dataset is from the LIBSVM data repository. The data has been preprocessed and scaled to a range of negative one to one for ease of use. 

The dataset consists of thirteen attributes including the class attribute. These attributes have been labeled as Attribute One, Attribute Two, Attribute Three, and so on up to Attribute Thirteen. 

Please cite this dataset as follows: Dataset from the LIBSVM data repository.
