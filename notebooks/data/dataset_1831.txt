QSAR-TID-101220 Dataset Information

This dataset contains QSAR data showing activity values of several compounds on drug target TID: 101220. The dataset has seven hundred nine rows and one thousand twenty-six features, including IDs and class feature.

The features represent FCFP six one hundred twenty-four bit molecular fingerprints, which were generated from SMILES strings using the Pipeline Pilot program, Dassault Systèmes BIOVIA.

Demographic information includes molecule ChEMBL ID and median PX fifty.

Molecule ChEMBL ID is a unique identifier for each molecule.

Median PX fifty is the median of the pseudopIC fifty values for each molecule.

FCFP six one hundred twenty-four bit molecular fingerprints are a type of molecular descriptor used to characterize the structure of molecules.

Pipeline Pilot is a software program used to generate molecular fingerprints from SMILES strings.

Dassault Systèmes BIOVIA is the company that developed the Pipeline Pilot program.

Keywords: QSAR, drug target, molecular fingerprints, SMILES strings, Pipeline Pilot, Dassault Systèmes BIOVIA, molecule ChEMBL ID, median PX fifty.
