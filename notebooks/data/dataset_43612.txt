Wine Dataset 

Tags: Computer Systems, Machine Learning 

These data are the results of a comprehensive chemical analysis of wines grown in the same region in Italy, but derived from three distinct cultivars. The analysis meticulously determined the quantities of various constituents found in each of the three types of wines.

The classification of these wines into their respective categories is the primary task associated with this dataset.

Source: UCI Machine Learning Repository

Features: 
Alcohol Content, Acid Levels, Ash Content, Alkalinity of Ash, Magnesium Levels, Total Phenol Content, Flavonoid Levels, Non-Flavonoid Phenol Levels, Proanthocyanin Levels, Colour Intensity, Hue Levels, and Protein Levels, along with the Class Label.

Keywords: Wine Analysis, Chemical Composition, Classification, Machine Learning, Italian Wines
