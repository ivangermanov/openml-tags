The Cervical Cancer Risk Factors Data Set was collected at the Hospital Universitario de Caracas in Caracas, Venezuela. This data set comprises demographic information, habits, and historic medical records of patients. Unfortunately, several patients decided not to answer some of the questions due to privacy concerns, resulting in missing values.

### Attribute Information

This data set includes attributes such as age, number of sexual partners, age of first sexual intercourse, number of pregnancies, smoking habits, use of hormonal contraceptives, intrauterine device use, history of sexually transmitted diseases, and diagnostic results.

### Data Set Description

The data set contains a total of thirty-two attributes, including demographic information, habits, and historic medical records of patients. Some patients chose not to disclose certain information, resulting in missing values. The data set aims to identify potential risk factors for cervical cancer.

### Features

Age, Number of Sexual Partners, Age of First Sexual Intercourse, Number of Pregnancies, Smoking Habits, Use of Hormonal Contraceptives, Intrauterine Device Use, History of Sexually Transmitted Diseases, Diagnostic Results

### Target Variables

Hinselmann, Schiller, Cytology, Biopsy
