Timing Attack Dataset Created on September Seventeenth Two Thousand Twenty-Two

Tags: Bleichenbacher Timing Attack

This dataset contains information about a timing attack with a two hundred fifty-six microsecond delay.

Attribute Descriptions:

The dataset includes attributes related to three types of messages: Change Cipher Spec, TLS Alert, and TCP Disconnect.

For each type of message, the attributes include:

* TCP source port
* TCP destination port
* TCP port
* TCP stream
* TCP segment length
* TCP sequence number
* TCP next sequence number
* TCP acknowledgment number
* TCP header length
* TCP flags (reserved, nonce, congestion window reduced, ECN echo, urgent, acknowledgment, push, reset, syn, fin)
* TCP window size value
* TCP calculated window size
* TCP window size scale factor
* TCP checksum status
* TCP urgent pointer
* TCP options (no operation, kind, length, timestamp value)
* TCP time delta
* Message order

The dataset also includes labels and vulnerable classes.
