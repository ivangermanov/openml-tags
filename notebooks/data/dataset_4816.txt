The HELOC dataset is part of the classification on numerical features benchmark and has been transformed in the same way as the other datasets used in the Tabular Data Benchmark. 

This dataset originates from the Home Equity Line of Credit dataset available on Kaggle, and users are requested to give credit to the original source when utilizing this dataset.

The HELOC dataset comprises various features, including external risk estimate, months since oldest trade open, average months in file, number of satisfactory trades, and percentage of trades never delinquent. Additionally, it includes months since most recent delinquency, maximum delinquency in public records last twelve months, total number of trades, and percentage of install trades. Other features include months since most recent inquiry excluding seven days, number of inquiries last six months, net fraction revolving burden, net fraction install burden, number of revolving trades with balance, and percentage of trades with balance.
