Asia Dataset 

Tags: Computer Systems, Machine Learning 

Dataset Description 
A synthetic dataset about lung diseases, including tuberculosis, lung cancer, or bronchitis, and visits to Asia. This dataset was originally created by Lauritzen and Spiegelhalter and has since been widely used.

Format of the Dataset 
The dataset is a data frame consisting of five thousand rows and eight binary variables. Each variable is represented by a binary value of zero or one, corresponding to yes or no.

The eight variables include Dyspnea, Tuberculosis, Lung Cancer, Bronchitis, Visit to Asia, Smoking, Chest X-Ray, and Tuberculosis versus Lung Cancer/Bronchitis.

Source 
This dataset is available online at bnlearn.com/bnrepository.

References 
This dataset was originally published in a journal article titled Local Computation with Probabilities on Graphical Structures and Their Application to Expert Systems.

Features 
Visit to Asia, Smoking, Tuberculosis, Lung Cancer, Bronchitis, Tuberculosis versus Lung Cancer/Bronchitis, Chest X-Ray, Dyspnea
