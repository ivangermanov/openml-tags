Features of this dataset include an X Box and Y Box which define the width and height of the box respectively. The On Pixel feature is also included. Additionally, there are various bar metrics such as X Bar, Y Bar, X Squared Bar, Y Squared Bar, XY Bar, X Squared Y Bar, and XY Squared Bar. Furthermore, Edge features including X Edge, X Edge Vertical Y, Y Edge, and Y Edge Vertical X are part of this dataset. Finally, the Class feature is also present.
