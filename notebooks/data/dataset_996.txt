Prnn F Glass Dataset 

Tags: Chemistry, Life Science, Myth Busting, Study 

The binarized version of the original dataset has been created by converting the multi-class target feature into a two-class nominal target feature. This conversion involved re-labeling the majority class as Positive and all other classes as Negative. 

Features of this dataset include Refractive Index, Sodium, Magnesium, Aluminum, Silicon, Potassium, Calcium, Barium, Iron, and Binary Class.
