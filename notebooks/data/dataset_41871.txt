FOREX USDDKK Hour Close

Tags: Finance, Forex, Forex Close, Forex Hour, Language, Text and Literature

Data Description:

This dataset contains the historical price data of the US Dollar versus Danish Krone (USD/DKK) Forex pair, obtained from Dukascopy Historical Data Feed. Each instance represents a single hourly candlestick, covering the period from January first to December thirteenth. Since Forex markets do not operate on weekends, these days are excluded from the dataset. The timestamp feature is set to the Europe/Amsterdam timezone.

The class attribute is determined by comparing the mean of the Close Bid and Close Ask prices for the current hour with the mean of the Close Bid and Close Ask prices for the subsequent hour. A value of true indicates an upward trend, whereas false signifies a downward or stagnant trend.

Attributes:

* Timestamp: The date and time of each data point, specified in the Europe/Amsterdam timezone.
* Bid Open: The opening bid price for each hourly interval.
* Bid High: The highest bid price recorded during each hourly interval.
* Bid Low: The lowest bid price recorded during each hourly interval.
* Bid Close: The closing bid price for each hourly interval.
* Bid Volume: The frequency of changes to the bid price within each hourly interval.
* Ask Open: The opening ask price for each hourly interval.
* Ask High: The highest ask price recorded during each hourly interval.
* Ask Low: The lowest ask price recorded during each hourly interval.
* Ask Close: The closing ask price for each hourly interval.
* Ask Volume: The frequency of changes to the ask price within each hourly interval.
* Class: A binary indicator of the future trend in the mean close price.
