Nintendo 3DS Games Dataset 

Machine Learning, Statistics 

Who does not love a good handheld video game? There are plenty of titles to explore, whether you are interested in data analytics or simply want to play through them if you own the console.

This dataset contains release and user review information for a vast collection of Nintendo 3DS games.

Acknowledgement goes to the team behind RAWG.IO for their hard work in creating an open-source repository of video game information and their wonderfully documented API service.

Features:

* Slug
* Play Time
* Release Date
* Rating
* Top Rating
* Number of Ratings
* Number of Reviews Text Count
* Added Date
* Metacritic Score
* Suggestions Count
* Last Updated
* Game Tags
* Number of Reviews
* Genres
* Yet to Play
* Owned
* Beaten
* To Play
* Dropped
* Currently Playing

Keywords: Nintendo 3DS, Handheld Video Games, Data Analytics, Release Information, User Reviews, RAWG.IO, API Service.
