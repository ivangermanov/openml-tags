Mamals Dataset for Image Classification (Stylized)

This dataset is categorized into various groups, each with a unique file name and file path. The style of each image is also identified, along with its style name and style path. Additionally, the stylized path of each image is provided.

Features: Category, File Name, File Path, Style, Style Name, Style Path, Stylized Path

Keywords: Image Classification, Mammals, Stylized Images, Dataset, File Paths
