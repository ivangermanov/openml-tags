**Masters Degrees Programs**

**Context**
I was searching for a master's degree program in data science when I stumbled upon an amazing website, Masters Portal. I decided to scrape the website to take my time analyzing all master's programs available around the world.

**Content**
This dataset contains a vast collection of master's degree programs from countries worldwide. The dataset is a treasure trove of information, providing insights into various aspects of master's programs.

**Features**
The dataset includes information about the country, university, program name, program type, deadline, duration, language, tuition fees, start date, IELTS score, academic requirements, and program structure. Additionally, it provides facts about the program, city, and program URL.

**Keywords:** Masters Programs, Data Science, University Rankings, Tuition Fees, Academic Requirements
