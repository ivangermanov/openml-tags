Tennis Major Tournament Match Statistics - Australian Open Women 2013 

This dataset is part of a collection of eight files containing match statistics for both women and men at the four major tennis tournaments of the year 2013. Each file has forty-two columns and a minimum of seventy-six rows. This file specifically contains the statistics for the Australian Open Women's Edition for 2013.

Attribute Information 

The dataset includes attributes such as the names of the two players participating in each match, the result of the match, and various performance metrics for each player. These performance metrics include first serve percentage, first serves won, second serve percentage, second serves won, aces, double faults, winners, unforced errors, break points created, break points won, net points attempted, net points won, total points won, and set results.

Result refers to the outcome of the match, where one indicates that Player One won the match and zero indicates that Player Two won the match. The set results are also included, with attributes for each player indicating their score in each set.

Features 

The dataset includes a wide range of features related to the performance of each player in the match. These features provide a comprehensive picture of each player's strengths and weaknesses during the tournament.
