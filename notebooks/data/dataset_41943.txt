**ILPD Numeric Dataset Tags**

The ILPD dataset from the OpenCC eighteen, with all categorical variables label encoded, is a comprehensive dataset featuring various attributes. This dataset is sourced from the original dataset, and users are requested to cite the authors of the original dataset.

The dataset comprises a range of features, including variables one through ten, culminating in a class feature. 

Keywords: ILPD, dataset, categorical variables, label encoded, OpenCC, attributes.
