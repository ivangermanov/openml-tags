Firewall Logs Dataset

Tags: Firewall Logs

Features: The Source Port from which the connection originated, the Destination Port to which the connection was directed, the NAT Source Port used for Network Address Translation, the NAT Destination Port used for Network Address Translation, the Action taken by the firewall, the total number of Bytes transferred, the number of Bytes Sent, the number of Bytes Received, the total number of Packets transferred, the Elapsed Time in seconds, the number of packets sent, and the number of packets received.

Keywords: Firewall, Network Security, Logs, Connection Details
